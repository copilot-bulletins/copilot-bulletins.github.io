Biweekly #1 · March 2 to March 16, 2026 · Devs Edition

🎯 This edition's thread

The past two weeks mark a turning point in Copilot's agentic capabilities. Both VS Code and JetBrains IDEs shipped major agent upgrades — hooks, plugins, memory sharing, custom agents — while GPT-5.4 landed with stronger multi-step reasoning. Copilot is moving from code assistant to programmable development partner.

🚀 Take action

  • Select GPT-5.4 in the model picker and test it on a complex coding task.
  • Update to VS Code v1.110 to unlock agent hooks, plugins, and browser tools.
  • JetBrains users: update the Copilot plugin for custom agents, sub-agents, and plan agent (now GA).
  • Try Copilot code review from your terminal with gh pr edit --add-reviewer @copilot`` — see the code review docs.
  • Questions or need support? Reach out to your GitHub team, we're here to help.

🏆 Top 5 updates

1. GPT-5.4 lands in GitHub Copilot

OpenAI's latest agentic coding model is now available across all Copilot clients — VS Code, JetBrains, Visual Studio, Xcode, Eclipse, GitHub.com, Mobile, CLI, and the coding agent. Early testing shows improved success rates on complex, multi-step tasks with enhanced logical reasoning.

→ Changelog: GPT-5.4 is generally available in GitHub Copilot

2. VS Code v1.110 supercharges your agent workflow

Agents get hooks for lifecycle automation, conversation forking for exploration, auto-approve via /yolo, plugin bundles, built-in browser tools, shared memory across Copilot surfaces, and manual context compaction with /compact. Long-distance next edit suggestions now predict edits anywhere in your file.

→ Changelog: GitHub Copilot in Visual Studio Code v1.110 – February release

3. JetBrains IDEs get a major agentic upgrade

Custom agents, sub-agents, and the plan agent are now GA. Agent hooks arrive in preview, along with MCP auto-approve support. The update also adds instruction file support (AGENTS.md, CLAUDE.md), a thinking panel for reasoning models, and context window usage indicators.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

4. Request Copilot code review straight from the CLI

Run gh pr edit --add-reviewer @copilot`` to add Copilot as a reviewer without leaving your terminal. The interactive mode also lists Copilot alongside teammates. This release adds search-based reviewer selection, dramatically improving performance in large organizations.

→ Changelog: Request Copilot code review from GitHub CLI

5. Figma MCP server bridges design and code in VS Code

Connect Copilot to Figma's MCP server to pull design context into code and send rendered UI back as editable Figma frames. This creates a continuous design-to-code-to-design loop, available to all Copilot subscribers with a Figma account.

→ Changelog: Figma MCP server can now generate design layers from VS Code
