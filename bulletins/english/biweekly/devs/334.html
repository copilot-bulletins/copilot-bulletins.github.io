🏆 Top 5 updates

1. JetBrains gets a major agentic upgrade

Custom agents, sub-agents, and plan agent are now GA in JetBrains IDEs. Agent hooks land in preview, MCP gains auto-approve, and new AGENTS.md support lets you tailor Copilot to your project's conventions — all from within your IDE.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

2. Request Copilot code review from GitHub CLI

No more switching to the browser for code reviews. With GitHub CLI v2.88.0, run gh pr edit --add-reviewer @copilot`` or select Copilot interactively during gh pr create. The release also brings faster, search-based reviewer selection for large organizations.

→ Changelog: Request Copilot code review from GitHub CLI

3. Skip CI approval for coding agent PRs

Copilot coding agent PRs previously required manual workflow approval, slowing the iteration loop. A new repository setting lets admins allow Actions workflows to run automatically on agent-pushed changes — useful for repos where faster feedback outweighs the security trade-off.

→ Changelog: Optionally skip approval for Copilot coding agent Actions workflows

4. Explore repositories visually in Copilot Chat on the web

Copilot Chat on github.com now shows a file tree alongside file contents. Browse the codebase, jump between files, and add them as references to your chat — all without leaving the conversation. Available in public preview.

→ Changelog: Explore a repository using Copilot on the web

5. GitHub Copilot SDK: embed agentic execution in your apps

The new Copilot SDK lets you embed the same planning and execution engine that powers Copilot CLI directly into your own applications. Define intent and constraints, and let the agent plan steps, invoke tools, and adapt at runtime — via MCP and structured context.

→ Blog: The era of "AI as text" is over. Execution is the new interface.
