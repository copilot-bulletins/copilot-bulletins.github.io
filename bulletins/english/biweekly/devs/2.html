🏆 Top 5 updates

1. Copilot coding agent now starts work 50% faster

The background coding agent now spins up 50% faster—meaning you get pull requests from scratch more quickly and iteration loops on existing PRs shrink noticeably. Whether you're assigning issues or mentioning @copilot in a PR comment, the wait before Copilot gets to work is cut in half.

→ Changelog: Copilot coding agent now starts work 50% faster

2. GPT-5.4 mini now generally available

OpenAI's latest fast agentic coding model is rolling out to all paid Copilot plans. GPT-5.4 mini delivers the fastest time to first token in the lineup, excels at codebase exploration, and carries a 0.33x premium multiplier—making it a cost-efficient first choice for quick tasks and agent mode.

→ Changelog: GPT-5.4 mini is now generally available for GitHub Copilot

3. More visibility into Copilot coding agent sessions

Session logs now surface built-in and custom setup steps, show when subagents are spawned (collapsed by default), and make it easier to trace what Copilot is doing at each stage. Less guessing, faster debugging—verify environment setup and understand agent decisions without leaving the logs view.

→ Changelog: More visibility into Copilot coding agent sessions

4. Gemini 3.1 Pro available in JetBrains, Xcode, and Eclipse

Google's Gemini 3.1 Pro is now in public preview across JetBrains IDEs, Xcode, and Eclipse—on top of its existing availability on github.com, VS Code, Visual Studio, and GitHub Mobile. Admins on Business/Enterprise plans need to enable the model policy; once done, it appears in the chat model picker across agent, ask, and edit modes.

→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

5. Configure Copilot coding agent's validation tools

Repository admins can now choose exactly which validation tools the coding agent runs—CodeQL, secret scanning, GitHub Advisory Database, and Copilot code review are all on by default and free (no GHAS license required). Disable specific checks per repo from the Copilot → Coding agent section in repository settings.

→ Changelog: Configure Copilot coding agent's validation tools
