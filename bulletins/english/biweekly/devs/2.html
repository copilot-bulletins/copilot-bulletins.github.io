🏆 Top 5 updates

1. Major agentic capability upgrades land in JetBrains IDEs

Custom agents, sub-agents, and plan agent are now GA in JetBrains IDEs, with agent hooks in public preview. MCP auto-approve, agent instruction files support, and auto model selection also land in this update — a comprehensive leap forward for JetBrains developers.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

2. Copilot coding agent now starts work 50% faster

Copilot coding agent now starts 50% faster. Whether you assign an issue, start from the Agents tab, or mention @copilot in a PR comment, you get a faster feedback loop — less wait before the agent begins writing and pushing code.

→ Changelog: Copilot coding agent now starts work 50% faster

3. GPT-5.4 mini is generally available in GitHub Copilot

GPT-5.4 mini, OpenAI's fastest agentic mini model, is now rolling out in GitHub Copilot across VS Code, JetBrains, Visual Studio, Xcode, Eclipse, GitHub.com, Mobile, and CLI. It excels at codebase exploration and grep-style tool use, with the fastest time to first token yet.

→ Changelog: GPT-5.4 mini is now generally available for GitHub Copilot

4. Gemini 3.1 Pro expands to JetBrains IDEs, Xcode, and Eclipse

Gemini 3.1 Pro expands to JetBrains IDEs, Xcode, and Eclipse in public preview, joining VS Code, Visual Studio, GitHub.com, and Mobile. Use it in agent, ask, and edit modes via the model picker. Business and Enterprise admins must enable the policy in org Copilot settings.

→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

5. GPT-5.3-Codex becomes Copilot's first LTS model

GPT-5.3-Codex is GitHub Copilot's first long-term support (LTS) model, guaranteed available for 12 months through February 4, 2027. It also becomes the new base model for Copilot Business and Enterprise from May 17, giving teams a stable, high-performing default for internal security reviews.

→ Changelog: GPT-5.3-Codex long-term support in GitHub Copilot
