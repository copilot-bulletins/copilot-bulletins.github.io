🏆 Top 5 updates

1. Copilot SDK now in public preview

The GitHub Copilot SDK puts the same production-tested agent runtime powering Copilot cloud agent and Copilot CLI in your hands. Available in Node.js, Python, Go, .NET, and Java, it lets you embed agentic workflows—tool invocation, streaming, and multi-turn sessions—directly into your own applications and services.

→ Changelog: Copilot SDK in public preview

2. Run multiple agents at once with /fleet in Copilot CLI

/fleet is a new Copilot CLI slash command that orchestrates multiple agents in parallel. Give it a goal with clearly separable tracks—refactoring, docs, tests—and the orchestrator decomposes the work, dispatches agents simultaneously, and synthesizes results. Ideal for large, multi-file changes that previously required sequential passes.

→ Blog: Run multiple agents at once with /fleet in Copilot CLI

3. Copilot in Visual Studio gets custom agents and agent skills

The March update for Visual Studio introduces custom agents as .agent.md files, reusable agent skills, enterprise MCP governance, and the find_symbol tool for language-aware navigation in agent mode. Profiling capabilities are also integrated with Copilot, enabling inline performance analysis while debugging.

→ Changelog: GitHub Copilot in Visual Studio — March update

4. Copilot cloud agent now signs every commit

Copilot cloud agent now produces verified, cryptographically signed commits. This removes a hard blocker for teams with "Require signed commits" branch protection rules, and gives developers confidence in the provenance of all agent-generated code appearing in their repositories.

→ Changelog: Copilot cloud agent signs its commits

5. Agent-driven development: lessons from the field

A GitHub Applied Science researcher shares how they built a benchmark analysis tool using Copilot CLI and the Copilot SDK—with five engineers shipping 11 new agents in under three days. The post distills practical strategies for prompting, architecture, and iteration that apply to any agent-first codebase.

→ Blog: Agent-driven development in Copilot Applied Science
