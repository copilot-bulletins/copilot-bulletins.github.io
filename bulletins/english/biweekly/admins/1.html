The pattern is clear: Copilot is becoming easier to manage, audit, and govern at scale.

🏆 Top 5 updates

1. GPT-5.3-Codex LTS: enterprise model stability, guaranteed

GPT-5.3-Codex is GitHub's first LTS model, guaranteed available for 12 months from launch (through February 4, 2027) for Copilot Business and Enterprise. Organizations now have the predictability they need for internal security reviews without rushing to adopt new models before internal approval processes are complete.

→ Changelog: GPT-5.3-Codex long-term support in GitHub Copilot

2. Accurate model reporting even when auto selection is on

Copilot usage metrics now resolve "Auto" model selections to the actual model used, giving admins a precise view of which models run across their org. This applies to the REST API and dashboard at enterprise, org, and user levels—supporting accurate compliance and audit reporting.

→ Changelog: Copilot usage metrics now resolve auto model selection to actual models

3. Configurable security validation for the coding agent

Repository admins can now configure which validation tools the coding agent runs—CodeQL, secret scanning, and Copilot code review are all on by default and free of charge. Teams with specific constraints (e.g., slow CodeQL scans) can disable individual checks directly from repository settings.

→ Changelog: Configure Copilot coding agent's validation tools

4. Track CLI adoption at the organization level

Org admins can now view GitHub Copilot CLI activity in 1-day and 28-day usage reports, completing coverage alongside enterprise and user-level metrics. This includes daily active CLI users, session counts, and token usage totals—enabling accurate adoption tracking and planning.

→ Changelog: Copilot usage metrics now includes organization-level GitHub Copilot CLI activity

5. Permanent audit trail from coding agent commits

Every commit from Copilot coding agent now includes an Agent-Logs-Url trailer, linking directly to the full session logs. Admins can trace any agent-authored change back to its session context, supporting code review processes, audit investigations, and compliance workflows.

→ Changelog: Trace any Copilot coding agent commit to its session logs
