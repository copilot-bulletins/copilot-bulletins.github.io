🏆 Top 5 updates

1. GPT-5.3-Codex Long-Term Support: Stability You Can Plan Around

GitHub commits to keeping GPT-5.3-Codex available for 12 months (through February 4, 2027) — a first-of-its-kind LTS pledge for enterprise customers. This gives organizations the runway needed for internal security and compliance reviews before adopting a new base model. The model also delivers a high code survival rate among enterprise users.

→ Changelog: GPT-5.3-Codex long-term support in GitHub Copilot

2. Copilot Coding Agent: 50% Faster and Smarter

The cloud-based coding agent now starts work 50% faster — cutting time from task to draft PR. A new semantic code search tool further reduces completion time by locating code by meaning, not exact syntax. Together, they deliver measurably faster ROI from agentic workflows.

3. Full Model Visibility in Usage Metrics — Even with Auto Selection

Copilot usage metrics now resolve auto model selections to the actual model used, across all report levels — enterprise, org, and user. This closes a key visibility gap for teams tracking AI consumption, enabling more precise ROI reporting and compliance documentation.

→ Changelog: Copilot usage metrics now resolve auto model selection to actual models

4. Gemini 3.1 Pro Now Available Across All Major IDEs

Google's Gemini 3.1 Pro is now in public preview across all major IDEs — JetBrains, Xcode, and Eclipse — in addition to VS Code and GitHub.com. This breadth reinforces GitHub's multi-model strategy, giving teams the flexibility to choose the right model for any task.

→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

5. Built-In Security Validation in the Coding Agent — No Extra License Needed

Copilot coding agent automatically runs CodeQL, secret scanning, GitHub Advisory Database checks, and code review before requesting your review — all free, with no GitHub Advanced Security license required. Organizations can configure which validations run per repository, balancing security coverage with delivery speed.
