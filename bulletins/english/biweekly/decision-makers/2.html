🏆 Top 5 updates

1. Copilot cloud agent can now research, plan, and iterate before coding

Copilot cloud agent can now research a codebase, generate an implementation plan for human review, and code — all before creating a pull request. Organizations can delegate broader development tasks to Copilot with approval gates at each stage, expanding the ROI of AI well beyond assisted coding.

→ Changelog: Research, plan, and code with Copilot cloud agent

2. Copilot organization custom instructions are generally available

Org-wide custom instructions for Copilot are now GA after a year in preview. Business and Enterprise admins can define consistent Copilot behavior across all repositories — covering Chat, code review, and cloud agent sessions — delivering standardized AI governance at the organizational level.

→ Changelog: Copilot organization custom instructions are generally available

3. Organization-level runner controls for Copilot cloud agent

Org admins can now set a default Actions runner for all cloud agent sessions and lock it so individual repositories cannot override it. Enforce larger runners for performance or self-hosted runners for internal resource access — consistent, centralized control across every team.

→ Changelog: Organization runner controls for Copilot cloud agent

4. Organization firewall settings for Copilot cloud agent

The cloud agent's network firewall, designed to prevent data exfiltration and prompt injection, can now be managed centrally at the org level. Admins can enforce policies, manage allowlists, and restrict per-repo overrides — making enterprise-wide Copilot cloud agent deployment safer and more consistent.

→ Changelog: Organization firewall settings for Copilot cloud agent

5. Copilot cloud agent now signs every commit

Copilot cloud agent cryptographically signs all commits it makes, displayed as Verified on GitHub. This removes a key deployment blocker: the agent now works in repositories with "Require signed commits" branch protection rules, expanding its reach into regulated and compliance-sensitive environments.

→ Changelog: Copilot cloud agent signs its commits
