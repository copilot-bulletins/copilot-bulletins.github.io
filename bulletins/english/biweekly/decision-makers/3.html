🏆 Top 5 updates

1. GitHub Commits to Enterprise Stability with Its First-Ever LTS Model

GitHub introduced GPT-5.3-Codex as its first long-term support (LTS) model for Copilot Business and Enterprise, guaranteeing availability through February 4, 2027. This 12-month commitment directly addresses enterprise procurement cycles by giving security and compliance teams the runway they need for internal reviews.

→ Changelog: GPT-5.3-Codex long-term support in GitHub Copilot

2. Copilot Coding Agent Starts Work 50% Faster

The coding agent now spins up 50% faster, reducing time between assigning a task and seeing the first pull request. For organizations already delegating issues and PRs to Copilot, this directly compounds developer throughput and shortens iteration cycles across the backlog.

→ Changelog: Copilot coding agent now starts work 50% faster

3. Every AI Commit Is Now Auditable — End to End

Commits authored by Copilot coding agent now include an Agent-Logs-Url trailer linking directly to the full session logs. Security and compliance teams can trace any AI-generated change back to the exact task, prompt, and execution context without leaving the pull request workflow.

→ Changelog: Trace any Copilot coding agent commit to its session logs

4. GPT-5.4 mini Now Generally Available Across All Copilot Plans

OpenAI's latest fast model, GPT-5.4 mini, is rolling out to all Copilot plans (Pro, Pro+, Business, Enterprise). Designed for the fastest time-to-first-token and stronger codebase exploration, it broadens the model portfolio available to every organization at a 0.33x premium request multiplier.

→ Changelog: GPT-5.4 mini is now generally available for GitHub Copilot

5. Copilot Usage Metrics Now Show Actual Models in Use

Usage metrics now resolve "Auto" selections to actual model names. Admins get an accurate, compliance-ready view of exactly which AI models are running across their organization — including breakdowns by feature, chat mode, and team level through the REST API.

→ Changelog: Copilot usage metrics now resolve auto model selection to actual models
