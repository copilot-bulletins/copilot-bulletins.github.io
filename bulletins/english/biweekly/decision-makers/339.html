🏆 Top 5 updates

1. 60 million Copilot code reviews reached

Copilot code review has grown 10X since launch, now powering one in five code reviews on GitHub. Over 12,000 organizations run it automatically on every pull request. WEX reports ~30% more code shipped after making Copilot code review a default across all repositories.

2. GitHub Copilot SDK brings agentic execution to your applications

The new Copilot SDK lets teams embed the same planning and execution engine that powers Copilot CLI into their own applications. This positions Copilot as programmable AI infrastructure, not just an IDE tool, enabling agentic workflows inside SaaS platforms, internal tools, and event-driven systems.

→ Blog: The era of "AI as text" is over. Execution is the new interface.

3. Security architecture for AI agents published

GitHub detailed the layered security architecture behind Agentic Workflows: container isolation, zero-secret agents, staged and vetted writes, and comprehensive logging. For organizations evaluating AI agent adoption, this provides the trust and governance framework needed to deploy agents safely at scale.

→ Blog: Under the hood: Security architecture of GitHub Agentic Workflows

4. Copilot agentic capabilities reach GA in JetBrains

Custom agents, sub-agents, and plan agent are now generally available in JetBrains IDEs. Agent hooks are in preview, and auto-approve for MCP tools reduces manual intervention. This extends Copilot's agentic maturity across the full IDE ecosystem, broadening enterprise reach.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

5. Auto model selection GA with built-in cost savings

Copilot auto model selection is generally available in JetBrains IDEs. It dynamically routes to the best-performing available model and offers a 10% discount on premium request multipliers, delivering both performance optimization and cost reduction for paid subscribers.
