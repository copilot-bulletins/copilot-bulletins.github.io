🏆 Top 5 updates

1. Copilot code review hits 60 million reviews

Usage has grown 10X since launch, with Copilot now handling one in five code reviews on GitHub. More than 12,000 organizations run automatic reviews, and customers like WEX report ~30% more code shipped after adoption.

→ Blog: 60 million Copilot code reviews and counting

2. The Copilot SDK brings agentic execution to your applications

The new GitHub Copilot SDK lets teams embed the same planning and execution engine behind Copilot directly into their own software. Applications can now delegate multi-step work to agents, ground execution in structured context via MCP, and operate outside the IDE.

→ Blog: The era of "AI as text" is over. Execution is the new interface.

3. Enterprise-grade security architecture for agentic workflows

GitHub published a deep dive into how Agentic Workflows enforce defense in depth: container-isolated agents with zero secret access, staged and vetted writes, firewall-controlled network egress, and comprehensive logging. A critical trust foundation for enterprise adoption.

→ Blog: Under the hood: Security architecture of GitHub Agentic Workflows

4. Agentic capabilities reach GA in JetBrains IDEs

Custom agents, sub-agents, and plan agent are now generally available in JetBrains IDEs. Agent hooks are in preview, auto model selection is GA with a 10% premium request discount, and AGENTS.md instruction files are supported for team-level customization.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

5. Coding agent can now skip workflow approval for faster iteration

A new repository setting lets administrators skip the manual approval step for GitHub Actions workflows triggered by Copilot coding agent. This accelerates the feedback loop for validating agent-generated code while keeping the secure default intact.

→ Changelog: Optionally skip approval for Copilot coding agent Actions workflows
