GitHub Copilot Bulletin — Monthly #1

March 2026 · Devs Edition

🎯 This edition's thread

This month, GitHub Copilot's agentic capabilities matured across every developer surface — IDEs, CLI, code review, and the coding agent. Agent hooks, shared memory, and a new agentic architecture for code review give developers more programmable control over AI-assisted workflows, while GPT-5.4 and bidirectional MCP integrations expand what agents can accomplish autonomously.

⚠️ Heads up

  • Upcoming deprecation of Gemini 3 Pro and GPT-5.1 models — Deadline: March 26 (Gemini 3 Pro) / April 1 (GPT-5.1 family) — Switch to Gemini 3.1 Pro or GPT-5.3-Codex in your model settings.

🚀 Take action

  • Select GPT-5.4 in the model picker in your IDE or on github.com to try OpenAI's latest agentic model.
  • Request a Copilot code review from the terminal — upgrade to GitHub CLI v2.88.0 and run gh pr edit --add-reviewer @copilot``.
  • Set up agent hooks in your repository to automate linting, enforce policies, or run custom commands during agent sessions.
  • Connect the Figma MCP server to VS Code to pull design context into code and push rendered UI back to Figma.
  • Questions or need support? Reach out to your GitHub team, we're here to help.

🏆 Top 5 updates of the month

1. VS Code v1.110 — Agents level up

Agent hooks let you run custom code at key lifecycle events. Fork conversations, queue follow-ups mid-run, and auto-approve with /yolo. Agent plugins, shared memory, context compaction, and browser tools make this the most agent-forward VS Code release yet.

→ Changelog: GitHub Copilot in Visual Studio Code v1.110 – February release

2. GPT-5.4 arrives in GitHub Copilot

OpenAI's latest agentic coding model is now GA across VS Code, JetBrains, Visual Studio, Xcode, Eclipse, github.com, GitHub CLI, and the coding agent. It delivers stronger logical reasoning and multi-step task execution for complex coding workflows.

→ Changelog: GPT-5.4 is generally available in GitHub Copilot

3. Copilot code review goes agentic — and comes to the CLI

Code review now runs on a tool-calling architecture, retrieving broader repository context for higher-signal, lower-noise feedback. You can also request a Copilot review from the terminal with gh pr edit --add-reviewer @copilot`` — no browser needed.

Sources:
→ Changelog: Copilot code review now runs on an agentic architecture
→ Changelog: Request Copilot code review from GitHub CLI

4. Figma MCP server: design-to-code round trip

The Figma MCP server now supports bidirectional workflows in VS Code — pull design context into code and push rendered UI back to Figma as editable frames. A seamless iterate-in-place loop for frontend developers building UI.

→ Changelog: Figma MCP server can now generate design layers from VS Code

5. JetBrains IDEs get custom agents, hooks, and more

Custom agents, sub-agents, and the plan agent are now GA. Agent hooks enter public preview alongside MCP auto-approve. New support for AGENTS.md instruction files, a thinking panel for reasoning models, and a context window indicator round out this major release.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs
