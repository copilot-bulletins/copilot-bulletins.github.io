GitHub Copilot Bulletin — Monthly (Long) #1

Marzo 2026 · Admins Edition

🎯 El hilo de esta edición

Este mes marca un punto de inflexión para la gobernanza de IA en GitHub. Enterprise AI Controls y el agent control plane alcanzaron disponibilidad general, junto con Copilot usage metrics, ofreciendo a los administradores una base sólida de visibilidad y control sobre la actividad agéntica. En paralelo, la plataforma multi-agente se expandió con Claude y Codex para todos los planes de pago, mientras que code review adoptó una arquitectura agéntica. El mensaje para los administradores es claro: las herramientas de gobernanza están listas para escalar la adopción de IA con confianza.

⚠️ Atención

  • Upcoming deprecation of Gemini 3 Pro and GPT-5.1 models — Fecha límite: 26 de marzo de 2026 (Gemini 3 Pro) y 1 de abril de 2026 (GPT-5.1, GPT-5.1-Codex, GPT-5.1-Codex-Mini, GPT-5.1-Codex-Max) — Verifique que los modelos alternativos (Gemini 3.1 Pro, GPT-5.3-Codex) estén habilitados en las políticas de modelos de Copilot antes de estas fechas.

🚀 Pasa a la acción

  • Configure las Enterprise AI Controls para gestionar agentes y políticas de IA a nivel enterprise.
  • Habilite GPT-5.4 en las políticas de modelos de Copilot de su organización.
  • Active el dashboard de Copilot usage metrics para monitorear adopción y uso.
  • Si usa self-hosted runners, complete la configuración para Copilot code review agéntico.
  • ¿Preguntas o necesita ayuda? Contacte a su equipo de GitHub, estamos aquí para ayudarle.

🏆 Top 8 de actualizaciones del mes

1. Enterprise AI Controls y agent control plane en disponibilidad general

Los administradores de enterprise cuentan ahora con un panel centralizado de AI Controls para gestionar políticas de agentes, monitorear sesiones agénticas en tiempo real y acceder a logs de auditoría detallados. Nuevos filtros permiten buscar sesiones por estado, repositorio y usuario.

Sources:
→ Changelog: Enterprise AI Controls & agent control plane now generally available
→ Changelog: Discover and manage agent activity with new session filters

2. Copilot usage metrics alcanza disponibilidad general

La suite completa de métricas de uso de Copilot es ahora GA, con dashboards interactivos, APIs a nivel enterprise/organización/usuario y telemetría expandida que incluye actividad de CLI, plan mode y nombres de usuario consistentes para Enterprise Managed Users.

Sources:
→ Changelog: Copilot metrics is now generally available
→ Changelog: Copilot usage metrics now includes enterprise-level GitHub Copilot CLI activity
→ Changelog: Copilot usage metrics now includes user-level GitHub Copilot CLI activity
→ Changelog: Copilot metrics now includes plan mode

3. Claude y Codex disponibles para Copilot Business y Pro

Claude de Anthropic y OpenAI Codex están ahora disponibles como coding agents para Copilot Business y Pro, sin suscripciones adicionales. Los administradores controlan la habilitación a nivel enterprise y organización, con gobernanza unificada a través del agent control plane.

→ Changelog: Claude and Codex now available for Copilot Business & Pro users

4. Copilot code review adopta arquitectura agéntica

Copilot code review utiliza ahora una arquitectura agéntica con tool-calling que explora el contexto del repositorio para ofrecer comentarios de mayor calidad y menor ruido. También se puede solicitar revisión de Copilot desde la CLI con gh pr edit --add-reviewer @copilot``.

Sources:
→ Changelog: Copilot code review now runs on an agentic architecture
→ Blog: 60 million Copilot code reviews and counting
→ Changelog: Request Copilot code review from GitHub CLI

5. API REST de Content Exclusion en preview pública

Los administradores de organización y enterprise pueden ahora gestionar reglas de exclusión de contenido de Copilot de forma programática mediante la nueva API REST, facilitando la automatización y gobernanza a escala con operaciones GET y SET.

→ Changelog: Copilot Content Exclusion REST API in public preview

6. GPT-5.4 disponible en GitHub Copilot

GPT-5.4, el último modelo agéntico de OpenAI, está disponible en todos los planes de pago de Copilot. Los administradores de Copilot Business y Enterprise deben habilitar la política de GPT-5.4 en la configuración de Copilot para que sus usuarios puedan acceder al modelo.

→ Changelog: GPT-5.4 is generally available in GitHub Copilot

7. Nuevos controles operativos para el coding agent

Los administradores de repositorio pueden ahora omitir la aprobación manual de workflows de GitHub Actions para el coding agent, acelerando el ciclo de feedback. Además, los cambios de configuración de red con enrutamiento basado en suscripción ya están en efecto para self-hosted runners.

Sources:
→ Changelog: Optionally skip approval for Copilot coding agent Actions workflows
→ Changelog: Network configuration changes for Copilot coding agent now in effect

8. Copilot coding agent para Jira en preview pública

La integración permite asignar issues de Jira directamente a Copilot coding agent. El agente analiza el contexto del issue, implementa cambios y abre un draft pull request en GitHub, todo bajo los controles de gobernanza existentes del enterprise.

→ Changelog: GitHub Copilot coding agent for Jira is now in public preview
