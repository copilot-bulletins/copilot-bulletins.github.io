GitHub Copilot Bulletin — Monthly (Long) #1

Marzo 2026 · Decision-Makers Edition

🎯 El hilo de esta edición

Este mes marca un punto de inflexión: GitHub Copilot se consolida como una plataforma multiagente con gobernanza empresarial completa. La llegada de Claude y Codex para todos los planes de negocio, junto con los Enterprise AI Controls y Copilot Metrics en disponibilidad general, permite a las organizaciones escalar la adopción de IA con visibilidad total y control operativo. Al mismo tiempo, el SDK de Copilot y la nueva arquitectura de seguridad para flujos agénticos posicionan a GitHub como la infraestructura sobre la que se construye el desarrollo de software del futuro.

⚠️ Atención

  • Upcoming deprecation of Gemini 3 Pro and GPT-5.1 models — Deadline: 26 de marzo (Gemini 3 Pro) y 1 de abril de 2026 (GPT-5.1, GPT-5.1-Codex, GPT-5.1-Codex-Mini, GPT-5.1-Codex-Max) — Actualicen sus flujos de trabajo a los modelos alternativos recomendados y verifiquen las políticas de modelos en la configuración de Copilot.

🚀 Pasa a la acción

  • Habilita GPT-5.4 en las políticas de modelos de Copilot de tu empresa para que tus equipos accedan al modelo de agentes más avanzado.
  • Activa los agentes Claude y Codex desde los AI Controls de tu empresa para ofrecer a tus desarrolladores una plataforma multiagente completa.
  • Explora los dashboards de Copilot usage metrics para medir la adopción y el impacto de la IA en tu organización.
  • Configura la integración de Copilot coding agent con Jira para acelerar la entrega desde los flujos de trabajo existentes de tu equipo.
  • ¿Preguntas o necesitas ayuda? Contacta a tu equipo de GitHub, estamos aquí para ayudarte.

🏆 Top 8 de actualizaciones del mes

1. Claude, Codex y Copilot: tres agentes bajo una misma plataforma

Claude de Anthropic y OpenAI Codex están ahora disponibles como coding agents para todos los planes de Copilot Business y Pro, sin suscripción adicional. Los tres agentes comparten contexto, memoria y gobernanza unificada, consolidando GitHub como la plataforma única para desarrollo asistido por IA.

Sources:
→ Changelog: Claude and Codex now available for Copilot Business & Pro users
→ Changelog: Copilot coding agent model picker for Copilot Business and Enterprise

2. Enterprise AI Controls y plano de control de agentes en GA

Los Enterprise AI Controls y el agent control plane alcanzan disponibilidad general, ofreciendo a los administradores un panel centralizado para gestionar políticas de IA, monitorizar sesiones de agentes, aplicar registros de auditoría y definir agentes personalizados a escala empresarial.

Sources:
→ Changelog: Enterprise AI Controls & agent control plane now generally available
→ Changelog: Discover and manage agent activity with new session filters

3. 60 millones de revisiones de código: Copilot como estándar de calidad

Copilot ha realizado más de 60 millones de revisiones de código, representando ya una de cada cinco revisiones en GitHub. La nueva arquitectura agéntica mejora la calidad del feedback, priorizando hallazgos de alto impacto sobre el volumen. Más de 12,000 organizaciones lo ejecutan automáticamente en cada pull request.

Sources:
→ Blog: 60 million Copilot code reviews and counting
→ Changelog: Copilot code review now runs on an agentic architecture

4. GPT-5.4: el modelo de agentes más avanzado llega a Copilot

GPT-5.4 de OpenAI, su modelo agéntico más reciente, ya está disponible en Copilot para planes Pro, Pro+, Business y Enterprise. Ofrece razonamiento lógico avanzado y ejecución de tareas complejas de múltiples pasos, disponible en todos los IDEs, github.com y GitHub Mobile.

Sources:
→ Changelog: GPT-5.4 is generally available in GitHub Copilot
→ Changelog: Copilot auto model selection is generally available in JetBrains IDEs

5. Copilot Metrics en GA: visibilidad total para medir el impacto

Copilot usage metrics alcanza disponibilidad general con dashboards de uso, generación de código y métricas de pull request a nivel de empresa, organización y usuario. Incluye APIs exportables para integración con herramientas BI y soporte de residencia de datos.

Sources:
→ Changelog: Copilot metrics is now generally available
→ Changelog: Organization-level Copilot usage metrics dashboard available in public preview
→ Changelog: Pull request throughput and time to merge available in Copilot usage metrics API

6. Copilot coding agent se integra con Jira

El coding agent de Copilot puede ahora recibir tareas directamente desde Jira. Al asignar un issue de Jira a Copilot, este analiza la descripción, implementa cambios y abre un pull request en borrador, reduciendo el cambio de contexto y acelerando la entrega sin abandonar los flujos existentes.

→ Changelog: GitHub Copilot coding agent for Jira is now in public preview

7. GitHub Copilot SDK: IA agéntica integrada en cualquier aplicación

El nuevo Copilot SDK permite integrar el mismo motor de planificación y ejecución agéntica que impulsa GitHub Copilot directamente en aplicaciones propias. Las organizaciones pueden delegar tareas complejas a agentes dentro de sus sistemas, utilizando MCP para contexto estructurado en tiempo real.

→ Blog: The era of "AI as text" is over. Execution is the new interface.

8. Seguridad empresarial para flujos de trabajo agénticos

GitHub ha publicado en detalle la arquitectura de seguridad de los Agentic Workflows: aislamiento por contenedores, cero acceso a secretos, escrituras validadas y registros exhaustivos. Este modelo ofrece garantías concretas para organizaciones que necesitan ejecutar agentes de IA con confianza.

→ Blog: Under the hood: Security architecture of GitHub Agentic Workflows
