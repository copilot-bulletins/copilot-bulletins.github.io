🎯 El hilo de esta edición

Este mes, GitHub Copilot consolida su transición de asistente de código a plataforma de IA empresarial. Con 60 millones de revisiones de código como hito de adopción, el lanzamiento del SDK para integrar ejecución agente en aplicaciones propias, y controles de gobernanza más maduros, la propuesta de valor para las organizaciones se amplía significativamente más allá del editor.

🏆 Top 5 de actualizaciones del mes

1. 60 millones de revisiones de código con Copilot

Copilot code review alcanza los 60 millones de revisiones, representando ya 1 de cada 5 revisiones de código en GitHub. Más de 12.000 organizaciones lo ejecutan automáticamente en cada pull request. WEX reporta un ~30% más de código desplegado tras adoptarlo como estándar.

→ Blog: 60 million Copilot code reviews and counting

2. GPT-5.4 disponible en GitHub Copilot

El modelo más reciente de OpenAI llega a GitHub Copilot con capacidades superiores de razonamiento lógico y ejecución de tareas complejas. Disponible en todos los entornos — VS Code, JetBrains, github.com, CLI y Copilot coding agent. Los administradores de Business y Enterprise deben habilitarlo en la política de modelos.

→ Changelog: GPT-5.4 is generally available in GitHub Copilot

3. GitHub Copilot SDK: la IA como plataforma de ejecución

GitHub lanza el Copilot SDK, que permite integrar el mismo motor de planificación y ejecución agente que impulsa Copilot CLI directamente en aplicaciones propias. Este cambio arquitectónico transforma Copilot de asistente conversacional en infraestructura programable para flujos de trabajo agentes.

→ Blog: The era of "AI as text" is over. Execution is the new interface.

4. Copilot coding agent alcanza madurez empresarial

Copilot coding agent incorpora selector de modelos, auto-revisión con code review, escaneo de seguridad integrado, agentes personalizados y transferencia entre nube y CLI. Junto con los nuevos filtros de sesión en Enterprise AI Controls, las organizaciones ganan visibilidad completa sobre la actividad de agentes.

Sources:
→ Blog: What's new with GitHub Copilot coding agent
→ Changelog: Discover and manage agent activity with new session filters
→ Changelog: Optionally skip approval for Copilot coding agent Actions workflows

5. Expansión del ecosistema de IDEs y MCP

VS Code v1.110 introduce agent hooks, plugins de agentes y herramientas de navegador agente. JetBrains IDEs llevan a GA agentes personalizados, sub-agentes, plan agent y selección automática de modelo. La integración del servidor MCP de Figma conecta diseño y código en un flujo bidireccional.

Sources:
→ Changelog: GitHub Copilot in Visual Studio Code v1.110 – February release
→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs
→ Changelog: Figma MCP server can now generate design layers from VS Code
