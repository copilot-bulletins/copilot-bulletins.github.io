🏆 Top 5 de actualizaciones del mes

1. GitHub lanza el Copilot SDK: IA agéntica embebida en sus propias plataformas

El nuevo Copilot SDK (preview pública) permite a equipos de ingeniería embeber las capacidades agénticas de Copilot directamente en aplicaciones internas, plataformas y flujos de trabajo. Compatible con Node.js, Python, Go, .NET y Java. Incluye BYOK (Bring Your Own Key) para empresas que prefieren sus propias claves de API.

→ Changelog: Copilot SDK in public preview

2. Copilot cloud agent: gobierno y seguridad a escala de organización

Los administradores de organización ahora pueden establecer runners predeterminados, bloquear su configuración, gestionar el firewall del agente contra exfiltración de datos, y confiar en commits verificados del agente. Estas capacidades facilitan el despliegue del cloud agent a escala con las garantías de seguridad y cumplimiento que exigen los entornos empresariales.

Sources:
→ Changelog: Organization runner controls for Copilot cloud agent
→ Changelog: Organization firewall settings for Copilot cloud agent
→ Changelog: Copilot cloud agent signs its commits

3. /fleet en Copilot CLI: múltiples agentes en paralelo para mayor velocidad

La nueva capacidad /fleet permite al orquestador de Copilot CLI descomponer tareas complejas y asignarlas a múltiples subagentes trabajando en paralelo sobre distintos módulos o archivos simultáneamente. Una refactorización que tomaría horas puede completarse en minutos, multiplicando el retorno de inversión en proyectos de desarrollo a gran escala.

→ Blog: Run multiple agents at once with /fleet in Copilot CLI

4. Métricas de uso del Copilot CLI: visibilidad completa por usuario para el ROI

Las organizaciones ahora tienen acceso a datos granulares de adopción del Copilot CLI por usuario: sesiones, solicitudes, consumo de tokens y versión del cliente. Estos datos permiten fundamentar decisiones de inversión, identificar equipos donde la habilitación es necesaria y optimizar la asignación de costes de la licencia de Copilot.

→ Changelog: Copilot usage metrics now includes per-user GitHub Copilot CLI activity in organization reports

5. GitHub Copilot en Visual Studio: agentes personalizados y gobierno de MCP

La actualización de marzo introduce agentes personalizados definidos como archivos .agent.md en el repositorio, control centralizado de servidores MCP a nivel de organización, y nuevas herramientas de análisis de rendimiento con inteligencia de Copilot. La extensibilidad empresarial del IDE más utilizado por equipos .NET avanza de forma significativa.

→ Changelog: GitHub Copilot in Visual Studio — March update
