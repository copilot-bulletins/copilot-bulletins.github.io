🏆 Top 5 de actualizaciones del mes

1. Copilot cloud agent: investigar, planificar y codificar sin pull requests

Copilot cloud agent ya no se limita a abrir pull requests. Ahora puede trabajar en una rama sin crear PR, generar un plan de implementación para tu revisión antes de escribir una sola línea, e investigar en profundidad cualquier pregunta sobre tu codebase.

→ Changelog: Research, plan, and code with Copilot cloud agent

2. Copilot SDK en preview pública: agentes en tus propias aplicaciones

El Copilot SDK llega en preview pública para Node.js, Python, Go, .NET y Java. Incorpora el mismo motor agéntico que impulsa Copilot cloud agent y Copilot CLI en tus propias aplicaciones: herramientas personalizadas, streaming, sesiones multi-turno y soporte BYOK incluidos.

Sources:
→ Changelog: Copilot SDK in public preview
→ Blog: The era of "AI as text" is over. Execution is the new interface.

3. GPT-5.4 mini disponible de forma general en GitHub Copilot

GPT-5.4 mini, el último modelo rápido de OpenAI para Copilot, está ya disponible para planes Pro, Pro+, Business y Enterprise. Funciona en VS Code, JetBrains, Visual Studio, Xcode, GitHub Mobile y CLI, y destaca especialmente en exploración de código y herramientas tipo grep.

→ Changelog: GPT-5.4 mini is now generally available for GitHub Copilot

4. JetBrains IDEs: actualización agéntica mayor

Agentes personalizados, sub-agentes y plan agent llegan a GA en JetBrains. Los agent hooks entran en preview pública con soporte de eventos como preToolUse y postToolUse. Se incorpora auto-approve para MCP, compatibilidad con archivos AGENTS.md y CLAUDE.md, y auto model selection queda en GA para todos los planes.

Sources:
→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs
→ Changelog: Copilot auto model selection is generally available in JetBrains IDEs

5. /fleet en Copilot CLI: múltiples agentes en paralelo

/fleet descompone tu objetivo en tareas independientes y las ejecuta simultáneamente con múltiples sub-agentes. Cada agente trabaja en su propio contexto pero comparte el sistema de archivos, coordinado por un orquestador central. Ideal para refactorizaciones, migraciones de codebase y generación de documentación a gran escala.

→ Blog: Run multiple agents at once with /fleet in Copilot CLI
