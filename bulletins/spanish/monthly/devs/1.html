🎯 El hilo de esta edición

Este mes, GitHub Copilot amplía sus capacidades agénticas en todos los frentes: IDE, terminal y revisión de código. Los agentes ya no solo asisten — ahora planifican, ejecutan y aprenden de forma autónoma. Con nuevos modelos, plugins y memorias compartidas, el flujo de desarrollo se transforma en una colaboración continua entre el desarrollador y la IA.

🏆 Top 5 de actualizaciones del mes

1. GitHub Copilot en VS Code v1.110: agentes programables y extensibles

VS Code ahora permite programar agentes con hooks, auto-approve y plugins instalables. Los agentes comparten memoria con el coding agent y code review, compactan el contexto automáticamente y exploran el código base con sub-agentes en paralelo. Copilot CLI también se integra de forma nativa.

→ Changelog: GitHub Copilot in Visual Studio Code v1.110 – February release

2. GPT-5.4 ya disponible en GitHub Copilot

El nuevo modelo agéntico de OpenAI llega a todos los planes de pago. GPT-5.4 destaca en razonamiento lógico y ejecución de tareas complejas de múltiples pasos. Disponible en VS Code, JetBrains, Visual Studio, Xcode, Eclipse, github.com, GitHub Mobile y CLI.

→ Changelog: GPT-5.4 is generally available in GitHub Copilot

3. GitHub Copilot CLI: de preview a disponibilidad general

Copilot CLI se gradúa como agente de desarrollo completo en la terminal. Incluye modo plan, autopilot, agentes personalizados, plugins, hooks y soporte MCP nativo. Puedes delegar tareas al coding agent en la nube y cambiar de modelo a mitad de sesión.

→ Changelog: GitHub Copilot CLI is now generally available

4. La revisión de código de Copilot se vuelve agéntica

Copilot code review ahora usa una arquitectura agéntica que explora el repositorio para entender la lógica y la arquitectura de tu código. Además, ya puedes solicitar revisiones de Copilot directamente desde tu terminal con gh pr edit --add-reviewer @copilot``.

Sources:
→ Changelog: Copilot code review now runs on an agentic architecture
→ Changelog: Request Copilot code review from GitHub CLI

5. JetBrains: agentes personalizados GA y hooks en preview

Los agentes personalizados, sub-agentes y el agente plan son ahora GA en JetBrains. Se añaden hooks en preview, auto-approve para MCP, soporte para archivos de instrucciones AGENTS.md y CLAUDE.md, y un panel de razonamiento para modelos con thinking extendido.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs
