GitHub Copilot Bulletin — Monthly #1

Marzo 2026 · Admins Edition

🎯 El hilo de esta edición

Este mes, GitHub ha consolidado su oferta de gobernanza empresarial para IA. Con la disponibilidad general de AI Controls, métricas de uso de Copilot y nuevos agentes de codificación, los administradores tienen ahora las herramientas necesarias para supervisar, medir y gobernar la adopción de IA a escala en toda la organización.

⚠️ Atención

  • Upcoming deprecation of Gemini 3 Pro and GPT-5.1 models — Fecha límite: 26 de marzo (Gemini 3 Pro) y 1 de abril (GPT-5.1) — Verifique las políticas de modelos habilitadas en su organización y actualice a las alternativas recomendadas.
  • Network configuration changes for Copilot coding agent now in effect — Acción requerida — Si utiliza runners auto-hospedados, actualice su firewall con los nuevos endpoints de red por suscripción.

🚀 Pasa a la acción

  • Revise y configure las políticas de agentes y modelos desde AI Controls en la configuración de su empresa.
  • Explore los nuevos dashboards de métricas de uso de Copilot para supervisar la adopción en su organización.
  • Habilite Claude y Codex como agentes de codificación desde la configuración de Copilot de su organización.
  • Configure la revisión automática de código con Copilot en los repositorios de su organización.
  • ¿Preguntas o necesita soporte? Contacte a su equipo de GitHub, estamos aquí para ayudar.

🏆 Top 5 de actualizaciones del mes

1. Enterprise AI Controls y agent control plane en disponibilidad general

La suite de gobernanza empresarial para IA ya está disponible de forma general. Los administradores pueden gestionar políticas de agentes, supervisar sesiones activas con nuevos filtros (por estado, repositorio y usuario), y auditar toda la actividad agéntica desde una vista centralizada.

Sources:
→ Changelog: Enterprise AI Controls & agent control plane now generally available
→ Changelog: Discover and manage agent activity with new session filters

2. Copilot usage metrics en disponibilidad general

Las métricas de uso de Copilot ofrecen dashboards y APIs a nivel de empresa, organización y usuario. Incluyen ahora telemetría de Copilot CLI, modo plan y nombres de usuario consistentes para Enterprise Managed Users, proporcionando visibilidad completa sobre la adopción y el consumo.

Sources:
→ Changelog: Copilot metrics is now generally available
→ Changelog: Copilot usage metrics now includes enterprise-level GitHub Copilot CLI activity
→ Changelog: Copilot usage metrics now includes user-level GitHub Copilot CLI activity
→ Changelog: Copilot metrics now includes plan mode

3. Claude y Codex disponibles para Copilot Business y Pro

Los agentes de codificación Claude (Anthropic) y Codex (OpenAI) ya están disponibles para suscriptores Business y Pro sin costo adicional. Los administradores deben habilitarlos a nivel de empresa y organización desde AI Controls, con gobernanza unificada, memoria compartida y registro de auditoría.

→ Changelog: Claude and Codex now available for Copilot Business & Pro users

4. GPT-5.4 en disponibilidad general y gestión de políticas de modelos

GPT-5.4, el nuevo modelo de codificación agéntica de OpenAI, está disponible en todos los planes de pago. Los administradores de Copilot Enterprise y Business deben habilitar la política del modelo en la configuración de Copilot para que sus equipos puedan acceder a él.

→ Changelog: GPT-5.4 is generally available in GitHub Copilot

5. Copilot code review con arquitectura agéntica en disponibilidad general

La revisión de código de Copilot ahora utiliza una arquitectura agéntica con herramientas de contexto. Los equipos que usan runners auto-hospedados necesitan completar una configuración adicional. Con más de 12.000 organizaciones ejecutando revisiones automáticas, esta actualización mejora la precisión y reduce el ruido.

Sources:
→ Changelog: Copilot code review now runs on an agentic architecture
→ Blog: 60 million Copilot code reviews and counting

📊 El mes en cifras

  • 60 millones de revisiones de código realizadas por Copilot desde su lanzamiento.
  • 1 de cada 5 revisiones de código en GitHub son ahora realizadas por Copilot.
  • 12.000+ organizaciones ejecutan Copilot code review de forma automática en cada pull request.
  • 10x crecimiento en el uso de Copilot code review desde su lanzamiento inicial.

  • Optionally skip approval for Copilot coding agent Actions workflows — Nueva configuración para que los administradores de repositorios controlen la aprobación de workflows del coding agent.
  • Copilot metrics report URLs update — Actualización de endpoints de descarga de informes; revise las reglas de firewall si usa allowlists CDN.
  • Copilot metrics reports now return consistent usernames for Enterprise Managed Users — Valores de usuario consistentes en informes para EMU.
  • GitHub Mobile: Track coding agent progress in real time with Live Notifications — Notificaciones en tiempo real de sesiones de agentes en GitHub Mobile.
  • Copilot Memory now on by default for Pro and Pro+ users in public preview — Los administradores enterprise mantienen control total sobre la disponibilidad de la memoria vía políticas.
