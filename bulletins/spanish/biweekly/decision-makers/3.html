Biweekly #3 · Del 10 de marzo al 24 de marzo de 2026 · Decision-Makers Edition

🎯 El hilo de esta edición

Esta quincena marca un punto de inflexión en la madurez del agente de codificación de Copilot. Las actualizaciones combinan velocidad, observabilidad y trazabilidad: el agente arranca un 50% más rápido, cada commit vincula a sus registros de sesión, y las métricas de uso revelan ahora los modelos reales. GitHub amplía además su portafolio de modelos con Gemini 3.1 Pro en todos los entornos principales.

🚀 Pasa a la acción

  • Activa Gemini 3.1 Pro desde las políticas de Copilot en tu empresa para que los equipos puedan seleccionarlo desde el selector de modelos.
  • Revisa y habilita el agente de codificación de Copilot en tu organización para comenzar a automatizar tareas de desarrollo.
  • Consulta las métricas de uso de Copilot para obtener visibilidad precisa del consumo real de modelos de IA por equipo.
  • ¿Tienes preguntas o necesitas ayuda? El equipo de GitHub está aquí para acompañarte.

🏆 Top 5 de actualizaciones

1. Gemini 3.1 Pro llega a JetBrains, Xcode y Eclipse

Google Gemini 3.1 Pro está disponible en versión preliminar pública para planes Business, Enterprise, Pro y Pro+. Se suma a VS Code, Visual Studio, GitHub Mobile y github.com, ampliando el catálogo de modelos de IA que los equipos pueden seleccionar según su caso de uso.

→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

2. El agente de codificación de Copilot arranca un 50% más rápido

El tiempo de inicio del agente de codificación de Copilot se ha reducido a la mitad. Esto acelera la creación de pull requests desde cero y el ciclo de iteración con @copilot, reduciendo el tiempo de espera y aumentando el retorno de cada tarea delegada al agente autónomo.

→ Changelog: Copilot coding agent now starts work 50% faster

3. Las métricas de uso revelan el modelo real en selección automática

Con el modo automático activo, las métricas de uso de Copilot ahora muestran el modelo real utilizado en lugar de una etiqueta genérica "Auto". Los responsables de decisiones obtienen visibilidad precisa del consumo de modelos de IA en sus organizaciones, con soporte para auditorías y cumplimiento normativo.

→ Changelog: Copilot usage metrics now resolve auto model selection to actual models

4. Trazabilidad completa: cada commit del agente enlaza a sus registros de sesión

Cada commit generado por el agente de codificación incluye ahora un enlace directo a los registros de sesión del agente. Esto simplifica la revisión del código generado por IA y refuerza los flujos de auditoría empresarial, con trazabilidad permanente entre cada cambio y su contexto de ejecución.

→ Changelog: Trace any Copilot coding agent commit to its session logs

5. Mayor visibilidad en las sesiones del agente de codificación

Las sesiones del agente de codificación muestran ahora los pasos de configuración iniciales, la salida de entornos personalizados y la actividad de los subagentes delegados. La supervisión del trabajo autónomo de IA se vuelve más transparente y verificable para los equipos de ingeniería y cumplimiento.

→ Changelog: More visibility into Copilot coding agent sessions
