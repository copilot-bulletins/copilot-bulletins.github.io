🎯 El hilo de esta edición

Estas dos semanas marcan un salto en la plataforma agéntica de GitHub Copilot: el agente de codificación arranca un 50% más rápido, las capacidades multi-agente alcanzan disponibilidad general en JetBrains, y Gemini 3.1 Pro llega a todos los entornos. La trayectoria es clara: autonomía acelerada, trazabilidad completa y mayor visibilidad para la dirección.

🏆 Top 5 de actualizaciones

1. Gemini 3.1 Pro ya disponible en todos los entornos principales

GitHub amplía su oferta multimodelo con Gemini 3.1 Pro en vista previa pública para todos los planes de pago. Disponible en github.com, GitHub Mobile, VS Code, Visual Studio, JetBrains, Xcode y Eclipse, reforzando la estrategia de elección de modelo sin restricciones.

→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

2. El agente de codificación de Copilot arranca un 50% más rápido

GitHub ha reducido a la mitad el tiempo de arranque del agente de codificación. Desde la creación de pull requests hasta la iteración con @copilot, el ciclo de desarrollo se acelera, impactando directamente en la productividad de los equipos y el retorno sobre la inversión en IA.

→ Changelog: Copilot coding agent now starts work 50% faster

3. Capacidades agénticas avanzadas en disponibilidad general en JetBrains

Agentes personalizados, sub-agentes y el agente de planificación alcanzan disponibilidad general en JetBrains IDEs. Con selección automática de modelo en GA, hooks de agente en vista previa y soporte para AGENTS.md, Copilot se consolida como plataforma agéntica madura en entornos empresariales.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

4. Visibilidad completa del uso de modelos en las métricas de Copilot

La actividad etiquetada como "Auto" ahora muestra el modelo real utilizado, ofreciendo datos precisos en la API y el panel de métricas. Facilita la auditoría de cumplimiento, la planificación del presupuesto de IA y el seguimiento de la adopción por equipo.

→ Changelog: Copilot usage metrics now resolve auto model selection to actual models

5. Trazabilidad total: de cada commit del agente a los logs de sesión

Cada commit generado por el agente de codificación incluye ahora un enlace directo a los logs de sesión completos. Proporciona trazabilidad permanente para entender las decisiones del agente durante revisiones de código y auditorías, clave para equipos con requisitos de gobernanza estrictos.

→ Changelog: Trace any Copilot coding agent commit to its session logs

📌 También en esta edición

  • Optionally skip approval for Copilot coding agent Actions workflows — nuevo ajuste de repositorio para ejecutar workflows de Actions automáticamente cuando Copilot hace push, acelerando el ciclo de validación.
  • Copilot usage metrics now includes organization-level GitHub Copilot CLI activity — seguimiento completo de la adopción de Copilot CLI a nivel organizacional, junto a las métricas de IDE.
  • Request Copilot code review from GitHub CLI — los desarrolladores ahora pueden solicitar revisiones de código de Copilot directamente desde el terminal con gh pr edit --add-reviewer @copilot``.
  • Monitor Copilot coding agent logs live in Raycast — seguimiento en tiempo real de las sesiones del agente desde el lanzador Raycast, sin cambiar de contexto.
