GitHub Copilot Bulletin — Biweekly #1

Del 2 al 16 de marzo de 2026 · Devs Edition

🎯 El hilo de esta edición

Las últimas dos semanas consolidan la era agéntica de GitHub Copilot. VS Code y JetBrains recibieron actualizaciones que llevan los agentes de lo experimental a lo práctico: hooks, plugins, memoria compartida y auto-approve para MCP. Con GPT-5.4 disponible en todas las superficies y el servidor MCP de Figma, el flujo de trabajo del desarrollador se expande más allá del editor.

🚀 Pasa a la acción

  • Selecciona GPT-5.4 en el selector de modelos de tu IDE o activa la selección automática de modelo para que Copilot elija por ti.
  • Configura agent hooks en VS Code o JetBrains para automatizar políticas y linters durante sesiones agénticas.
  • Conecta el servidor MCP de Figma para generar código desde diseños y enviar UI renderizada de vuelta a Figma.
  • Solicita code review de Copilot desde la terminal con gh pr edit --add-reviewer @copilot`` — actualiza a GitHub CLI v2.88.0 o superior.
  • ¿Preguntas o necesitas soporte? Contacta a tu equipo de GitHub, estamos aquí para ayudar.

🏆 Top 5 de actualizaciones

1. VS Code v1.110: los agentes se vuelven programables

La release de febrero de VS Code transforma los agentes en herramientas controlables: hooks para ejecutar código en eventos clave, fork de conversaciones, plugins agénticos, herramientas de navegador integradas y compactación de contexto con /compact. Copilot CLI ahora viene integrado nativamente.

→ Changelog: GitHub Copilot in Visual Studio Code v1.110 – February release

2. GPT-5.4 disponible en GitHub Copilot

El modelo agéntico más reciente de OpenAI llega a Copilot con rendimiento superior en tareas multi-paso y razonamiento lógico avanzado. Disponible en VS Code, JetBrains, Visual Studio, Xcode, Eclipse, GitHub.com, CLI y Mobile.

→ Changelog: GPT-5.4 is generally available in GitHub Copilot

3. Mejoras agénticas mayores en JetBrains

Custom agents, sub-agents y plan agent pasan a GA. Se estrenan agent hooks en preview, auto-approve para MCP, soporte para archivos AGENTS.md y CLAUDE.md, y un panel de razonamiento para modelos extendidos. Edit mode queda marcado como deprecado.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

4. El servidor MCP de Figma genera capas de diseño desde VS Code

Ahora puedes generar código a partir de diseños de Figma y enviar UI renderizada de vuelta como frames editables. Un flujo bidireccional diseño-código que conecta a diseñadores y desarrolladores a través de Copilot.

→ Changelog: Figma MCP server can now generate design layers from VS Code

5. Solicita code review de Copilot desde GitHub CLI

Ya puedes pedir revisión de código a Copilot directamente desde la terminal con gh pr edit --add-reviewer @copilot`` o gh pr create. Incluye búsqueda mejorada de revisores que acelera el flujo en organizaciones grandes.

→ Changelog: Request Copilot code review from GitHub CLI
