Biweekly #2 · Del 10 al 24 de marzo de 2026 · Devs Edition

🎯 El hilo de esta edición

Las últimas dos semanas han estado marcadas por una ola de mejoras al agente de codificación de Copilot y la expansión del catálogo de modelos. La velocidad de inicio se redujo un 50%, la búsqueda semántica de código acelera la exploración de repositorios y varios entornos de desarrollo integrado reciben capacidades agénticas completas. Al mismo tiempo, nuevos modelos —GPT-5.4 mini y Gemini 3.1 Pro— llegan a todos los IDEs más populares.

⚠️ Atención

  • GPT-5.3-Codex long-term support in GitHub Copilot — Fecha límite: 17 de mayo de 2026 — GPT-5.3-Codex reemplazará automáticamente a GPT-4.1 como modelo base para todas las organizaciones Copilot Business y Enterprise; al tener un multiplicador de 1× (vs 0× de GPT-4.1), esto puede afectar el consumo de solicitudes premium. Revisa el impacto con tu administrador en la documentación de modelos.

🚀 Pasa a la acción

  • Activa GPT-5.4 mini o Gemini 3.1 Pro en el selector de modelos de tu IDE consultando la documentación de modelos disponibles.
  • Delega tus próximas tareas al agente de codificación de Copilot y aprovecha el nuevo inicio un 50% más rápido.
  • Solicita revisiones de código de Copilot directamente desde la terminal con gh pr edit --add-reviewer @copilot`` — más info en la documentación de Copilot code review.
  • Configura las herramientas de validación del agente (CodeQL, secret scanning, etc.) desde los ajustes de tu repositorio en la sección Copilot → Coding agent.
  • ¿Tienes preguntas o necesitas ayuda? Contacta a tu equipo de GitHub, estamos aquí para apoyarte.

🏆 Top 5 de actualizaciones

1. Mejoras agénticas importantes en GitHub Copilot para JetBrains IDEs

Esta actualización marca un antes y un después en JetBrains: agentes personalizados, sub-agentes y el agente de planificación son ahora GA; los agent hooks llegan en preview; auto-approve para MCP está disponible; y la selección automática de modelos es GA. Todo sin salir del IDE.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

2. El agente de codificación de Copilot arranca un 50% más rápido

GitHub optimizó el tiempo de arranque del agente de codificación en la nube, logrando una reducción del 50%. Tanto al crear pull requests desde cero como al pedir a Copilot que itere sobre uno existente con @copilot, el ciclo de retroalimentación es ahora significativamente más corto.

→ Changelog: Copilot coding agent now starts work 50% faster

3. Gemini 3.1 Pro disponible en JetBrains IDEs, Xcode y Eclipse

Gemini 3.1 Pro llega en preview público al selector de modelos en GitHub Copilot Chat de JetBrains IDEs, Xcode y Eclipse (además de VS Code, Visual Studio, github.com y GitHub Mobile). Disponible para planes Pro, Pro+, Business y Enterprise tras activación del administrador.

→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

4. GPT-5.4 mini ya es de disponibilidad general en GitHub Copilot

El modelo mini más rápido de OpenAI para tareas de codificación está ahora disponible para todos los planes Copilot (Pro, Pro+, Business y Enterprise). Destaca en exploración de repositorios y herramientas tipo grep, con el menor tiempo hasta el primer token de su categoría.

→ Changelog: GPT-5.4 mini is now generally available for GitHub Copilot

5. Solicita revisiones de código de Copilot desde la CLI de GitHub

Puedes añadir a Copilot como revisor directamente desde la terminal con gh pr edit --add-reviewer @copilot``, sin abrir el navegador. La selección interactiva de revisores ahora usa búsqueda incremental, lo que mejora drásticamente el rendimiento en organizaciones grandes.

→ Changelog: Request Copilot code review from GitHub CLI

📌 También en esta edición

  • Copilot coding agent works faster with semantic code search — El agente ahora usa búsqueda semántica de código para localizar contexto relevante sin necesidad de patrones exactos, reduciendo el tiempo de ejecución en un 2% adicional.
  • More visibility into Copilot coding agent sessions — Los logs del agente ahora muestran los pasos de configuración, el output de copilot-setup-steps.yml y la actividad de sub-agentes de forma más clara.
  • Trace any Copilot coding agent commit to its session logs — Cada commit del agente incluye un trailer Agent-Logs-Url que enlaza directamente con los logs de sesión, facilitando las revisiones de código y la auditoría.
  • Configure Copilot coding agent's validation tools — Los administradores de repositorio pueden activar o desactivar herramientas de validación (CodeQL, secret scanning, Copilot code review) desde los ajustes del repositorio.
  • Optionally skip approval for Copilot coding agent Actions workflows — Nueva opción de repositorio para que los workflows de GitHub Actions se ejecuten automáticamente sin aprobación humana previa, acelerando la iteración con el agente.
