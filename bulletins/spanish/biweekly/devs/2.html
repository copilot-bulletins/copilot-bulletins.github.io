Biweekly #2 · Del 10 al 24 de marzo de 2026 · Devs Edition

🎯 El hilo de esta edición

Esta quincena la evolución del agente de codificación de Copilot es la nota dominante: arranca un 50% más rápido, usa búsqueda semántica de código y estrena hooks de agente con soporte de instrucciones propias en JetBrains. Paralelamente, GPT-5.4 mini y Gemini 3.1 Pro amplían el catálogo de modelos en todos los IDEs.

⚠️ Atención

  • GPT-5.3-Codex se convierte en el modelo base para Copilot Business y Enterprise — Fecha límite: 17 de mayo de 2026 — A partir de esa fecha, GPT-5.3-Codex reemplazará a GPT-4.1 como modelo base por defecto en todas las organizaciones de Business y Enterprise. Revisa el impacto en tu configuración con la documentación de modelos base y LTS.

🚀 Pasa a la acción

  • Prueba GPT-5.4 mini seleccionándolo en el selector de modelos de Copilot en tu IDE.
  • Configura agent hooks para automatizar flujos de trabajo en tus sesiones de agente con la documentación de hooks (disponible en JetBrains en preview pública).
  • Solicita revisiones de código de Copilot directamente desde la terminal usando gh pr edit --add-reviewer @copilot`` — consulta la documentación de code review con Copilot.
  • ¿Tienes preguntas o necesitas apoyo? Tu equipo de GitHub está aquí para ayudarte.

🏆 Top 5 de actualizaciones

1. Grandes mejoras en capacidades agénticas en GitHub Copilot para JetBrains IDEs

El plugin de JetBrains trae una actualización mayor: agentes personalizados, sub-agentes y plan agent ahora son GA; hooks de agente en preview pública; soporte para archivos AGENTS.md y CLAUDE.md; y auto-aprobación para MCP. Una actualización esencial para usuarios de IntelliJ, PyCharm y compañía.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

2. El agente de codificación de Copilot arranca un 50% más rápido

Copilot coding agent ha reducido su tiempo de arranque a la mitad. Crear PRs desde cero o iterar sobre uno existente con @copilot es ahora notablemente más ágil, lo que acorta el bucle de feedback y hace más fluida la colaboración con el agente.

→ Changelog: Copilot coding agent now starts work 50% faster

3. GPT-5.4 mini ya está disponible en GitHub Copilot

GPT-5.4 mini, el nuevo modelo rápido de OpenAI optimizado para tareas agénticas, llega a Copilot en disponibilidad general. Ofrece el menor tiempo hasta el primer token entre los modelos mini y destaca en la exploración de bases de código con herramientas de tipo grep.

→ Changelog: GPT-5.4 mini is now generally available for GitHub Copilot

4. Gemini 3.1 Pro disponible en preview pública en JetBrains, Xcode y Eclipse

Gemini 3.1 Pro se suma al selector de modelos en JetBrains IDEs, Xcode y Eclipse (también disponible en github.com, VS Code, Visual Studio y móvil). Los administradores de Copilot Business y Enterprise deben habilitarlo en la configuración de políticas de Copilot.

→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

5. Explora un repositorio con Copilot directamente en la web

Ahora puedes navegar el árbol de archivos de un repositorio mientras chateas con Copilot en github.com. Los archivos que seleccionas se añaden automáticamente al chat como referencias, facilitando preguntas en contexto y reduciendo el tiempo de copiar y pegar fragmentos de código. Disponible en preview pública.

→ Changelog: Explore a repository using Copilot on the web
