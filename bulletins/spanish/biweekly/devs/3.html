Biweekly #3 · Del 10 al 24 de marzo de 2026 · Devs Edition

🎯 El hilo de esta edición

Las dos últimas semanas marcaron la maduración del ecosistema agéntico de Copilot: el agente de codificación arranca un 50% más rápido, JetBrains recibe capacidades agénticas completas en GA y la búsqueda semántica de código acelera las tareas en la nube. El catálogo de modelos se amplió con GPT-5.4 mini y Gemini 3.1 Pro, reforzando la flexibilidad del desarrollador.

⚠️ Atención

  • GPT-5.3-Codex long-term support in GitHub Copilot — Fecha límite: 17 de mayo de 2026 — GPT-5.3-Codex reemplazará automáticamente a GPT-4.1 como modelo base en todos los planes Copilot Business y Enterprise; el multiplicador cambia de 0× a 1×. Consulta el impacto en la documentación de modelos soportados.

🚀 Pasa a la acción

  • Selecciona GPT-5.4 mini o Gemini 3.1 Pro en el selector de modelos de tu IDE consultando la documentación de modelos disponibles.
  • Delega tus próximas tareas al agente de codificación de Copilot y aprovecha el nuevo arranque un 50% más rápido.
  • Solicita revisiones de código directamente desde la terminal con gh pr edit --add-reviewer @copilot`` — más información en la documentación de Copilot code review.
  • Configura agent hooks en tu repositorio para personalizar el comportamiento del agente con la documentación de hooks.
  • ¿Tienes preguntas o necesitas ayuda? Contacta a tu equipo de GitHub, estamos aquí para apoyarte.

🏆 Top 5 de actualizaciones

1. Mejoras agénticas importantes en GitHub Copilot para JetBrains IDEs

Custom agents, sub-agents y el agente de planificación pasan a GA en JetBrains; los agent hooks llegan en preview con soporte para AGENTS.md y CLAUDE.md. Además, el auto-approve para MCP y la selección automática de modelos son GA, consolidando un entorno agéntico completo sin salir del IDE.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

2. El agente de codificación de Copilot arranca un 50% más rápido

GitHub optimizó el tiempo de arranque del agente de codificación en la nube, logrando una reducción del 50%. Tanto al crear pull requests desde cero como al pedir a Copilot que itere sobre uno existente con @copilot, el ciclo de retroalimentación es significativamente más corto.

→ Changelog: Copilot coding agent now starts work 50% faster

3. GPT-5.4 mini ya disponible para todos los planes de GitHub Copilot

El modelo mini más rápido de OpenAI para tareas de codificación llega con disponibilidad general a los planes Pro, Pro+, Business y Enterprise. Destaca en exploración de repositorios y herramientas tipo grep, con el menor tiempo hasta el primer token de su categoría. Multiplicador de solicitudes premium: 0,33×.

→ Changelog: GPT-5.4 mini is now generally available for GitHub Copilot

4. Gemini 3.1 Pro disponible en JetBrains IDEs, Xcode y Eclipse

Gemini 3.1 Pro llega en preview público al selector de modelos en JetBrains IDEs, Xcode y Eclipse, completando así su presencia en todos los IDEs principales de GitHub Copilot. Disponible para planes Pro, Pro+, Business y Enterprise una vez que el administrador habilite la política correspondiente.

→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

5. Solicita revisiones de código de Copilot desde la CLI de GitHub

Añade a Copilot como revisor directamente desde la terminal con gh pr edit --add-reviewer @copilot``, sin abrir el navegador. Esta versión introduce también búsqueda incremental en la selección de revisores, mejorando el rendimiento en organizaciones grandes. Requiere GitHub CLI v2.88.0 o superior.

→ Changelog: Request Copilot code review from GitHub CLI

📌 También en esta edición

  • Copilot coding agent works faster with semantic code search — El agente usa búsqueda semántica para localizar código relevante sin necesidad de patrones exactos, completando tareas un 2% más rápido sin configuración adicional.
  • More visibility into Copilot coding agent sessions — Los logs del agente ahora muestran los pasos de configuración, la salida de copilot-setup-steps.yml y la actividad de sub-agentes de forma más clara y estructurada.
  • Trace any Copilot coding agent commit to its session logs — Cada commit del agente incluye un trailer Agent-Logs-Url que enlaza directamente con los logs de sesión, facilitando las revisiones de código y la auditoría.
  • Configure Copilot coding agent's validation tools — Los administradores de repositorios pueden activar o desactivar herramientas de validación (CodeQL, secret scanning, Copilot code review) desde los ajustes del repositorio en la sección Copilot → Coding agent.
  • Monitor Copilot coding agent logs live in Raycast — Sigue los logs del agente en tiempo real desde Raycast en macOS o Windows sin necesidad de abrir el navegador.
