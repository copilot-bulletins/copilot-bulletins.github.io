GitHub Copilot Bulletin — Biweekly #1

Del 2 al 16 de marzo de 2026 · Devs Edition

🎯 El hilo de esta edición

Las últimas dos semanas consolidaron la era agéntica de GitHub Copilot. VS Code y JetBrains recibieron mejoras sustanciales en hooks, plugins y gestión de contexto para agentes. GPT-5.4 llegó como nuevo modelo de referencia, el servidor MCP de Figma conecta diseño y código bidireccionalmente, y la CLI ahora permite solicitar revisiones de Copilot sin salir de la terminal.

🚀 Pasa a la acción

  • Selecciona GPT-5.4 en el selector de modelos de Copilot y pruébalo en tus flujos de trabajo agénticos.
  • Instala el servidor MCP de Figma para conectar diseño y código directamente en VS Code.
  • Actualiza GitHub CLI a v2.88.0 y solicita revisiones de Copilot con gh pr edit --add-reviewer @copilot``.
  • Configura hooks de agente para automatizar pasos clave en tus sesiones agénticas.
  • ¿Preguntas o necesitas ayuda? Contacta a tu equipo de GitHub, estamos aquí para ayudarte.

🏆 Top 5 de actualizaciones

1. VS Code v1.110: agentes más potentes con hooks, plugins y memoria compartida

El release de febrero transforma los agentes en herramientas prácticas para tareas complejas. Incluye hooks para automatizar en puntos clave, plugins de agente, herramientas de navegador agéntico, bifurcación de conversaciones y memoria compartida entre Copilot CLI y el coding agent.

→ Changelog: GitHub Copilot in Visual Studio Code v1.110 – February release

2. GPT-5.4 llega a GitHub Copilot

El último modelo agéntico de OpenAI ya está disponible en todos los IDEs, GitHub.com, CLI y Mobile. GPT-5.4 demuestra mejoras consistentes en razonamiento lógico y ejecución de tareas multi-paso, ideal para flujos de trabajo complejos.

→ Changelog: GPT-5.4 is generally available in GitHub Copilot

3. JetBrains: agentes personalizados, hooks y MCP auto-approve en GA

Los agentes personalizados, sub-agentes y el agente de plan son ahora GA en JetBrains. Llegan además hooks de agente en preview, auto-approve para MCP, soporte para archivos AGENTS.md y CLAUDE.md, y un panel de razonamiento para modelos extendidos.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

4. Figma MCP: genera capas de diseño desde VS Code

El servidor MCP de Figma ahora permite enviar UI renderizada como frames editables en Figma, además de importar contexto de diseño al código. Un flujo bidireccional que conecta diseño e implementación sin salir del editor.

→ Changelog: Figma MCP server can now generate design layers from VS Code

5. Revisiones de código con Copilot desde GitHub CLI

Ya puedes solicitar revisiones de Copilot directamente desde la terminal con gh pr edit --add-reviewer @copilot`` o durante gh pr create. La selección de revisores ahora usa búsqueda en tiempo real, optimizada para organizaciones grandes.

→ Changelog: Request Copilot code review from GitHub CLI
