GitHub Copilot Bulletin — Biweekly #1

Del 2 al 16 de marzo de 2026 · Admins Edition

🎯 El hilo de esta edición

Las últimas dos semanas refuerzan una tendencia clara: GitHub amplía los controles de gobernanza para agentes de IA a nivel empresarial. Nuevas configuraciones de repositorio, filtros avanzados en el panel de control de agentes y requisitos de habilitación de políticas para modelos dan a los administradores mayor visibilidad y control sobre cómo se despliegan y utilizan las capacidades agénticas de Copilot en sus organizaciones.

🚀 Pasa a la acción

  • Revisa la nueva configuración de aprobación automática de workflows para el coding agent en los ajustes de repositorio de Copilot y define la política adecuada para tus repositorios.
  • Habilita GPT-5.4 en las políticas de modelos de Copilot de tu organización o empresa.
  • Explora los nuevos filtros de sesiones de agentes en AI Controls para monitorear la actividad agéntica en tu empresa.
  • Configura revisiones automáticas de código con Copilot en tus repositorios u organización mediante rulesets.
  • ¿Preguntas o necesitas soporte? Contacta a tu equipo de GitHub, estamos aquí para ayudarte.

🏆 Top 5 de actualizaciones

1. Nueva configuración para omitir la aprobación de workflows del coding agent

Los administradores de repositorio ahora pueden permitir que los workflows de GitHub Actions se ejecuten automáticamente cuando Copilot coding agent crea o actualiza un pull request, sin requerir aprobación humana previa. Por defecto, la aprobación sigue siendo obligatoria para proteger tokens y secretos.

→ Changelog: Optionally skip approval for Copilot coding agent Actions workflows

2. Nuevos filtros de sesión en el panel de control de agentes

Enterprise AI Controls incorpora filtros por estado, repositorio y usuario para las sesiones de agentes, facilitando la supervisión y gestión de la actividad agéntica en toda la empresa. Estos filtros complementan los existentes por agente y organización.

→ Changelog: Discover and manage agent activity with new session filters

3. GPT-5.4 disponible en GitHub Copilot

GPT-5.4, el último modelo agéntico de OpenAI, ya está disponible en Copilot para planes Pro, Pro+, Business y Enterprise. Los administradores de Copilot Business y Enterprise deben habilitar la política de GPT-5.4 en los ajustes de Copilot para que sus usuarios accedan al modelo.

→ Changelog: GPT-5.4 is generally available in GitHub Copilot

4. VS Code v1.110: hooks y controles para agentes

La versión de febrero de VS Code introduce hooks para ejecutar comandos personalizados en eventos clave del agente, permitiendo aplicar políticas, validaciones automáticas y bloqueo de comandos. También incluye controles de auto-aprobación y sandboxing de terminal para acotar el comportamiento de los agentes.

→ Changelog: GitHub Copilot in Visual Studio Code v1.110 – February release

5. Mejoras agénticas en GitHub Copilot para JetBrains

Agentes personalizados, sub-agentes y plan agent pasan a disponibilidad general. Los agent hooks llegan en preview público y requieren que los administradores de Copilot Business/Enterprise habiliten la política de "Editor preview features". También se añade auto-aprobación configurable para servidores MCP.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs
