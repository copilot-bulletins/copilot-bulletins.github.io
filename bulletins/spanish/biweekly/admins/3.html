Esta quincena consolida dos avances clave para los administradores: la llegada de la residencia de datos y la certificación FedRAMP amplía el control sobre el cumplimiento normativo, mientras que nuevas métricas para el CLI, la revisión de código y el agente en la nube ofrecen una visibilidad más completa sobre la adopción de Copilot en la organización.

🏆 Top 5 de actualizaciones

1. Residencia de datos de Copilot en EE.UU. y UE, y cumplimiento FedRAMP

GitHub Copilot garantiza ahora que el procesamiento de inferencia permanezca en la región designada (EE.UU. o UE), cumpliendo además con FedRAMP Moderate para clientes del gobierno estadounidense. Los administradores habilitan la política desde la configuración de la empresa; las solicitudes tienen un incremento del 10% en el multiplicador premium.

→ Changelog: Copilot data residency in US + EU and FedRAMP compliance now available

2. Las métricas de uso incluyen recuentos de usuarios activos del agente en la nube

La API de métricas de uso incluye ahora recuentos de usuarios activos del agente en la nube en ventanas de 1, 7 y 28 días, a nivel de empresa y organización. Permite seguir la adopción del agente sin necesidad de agregar datos por usuario.

→ Changelog: Copilot usage metrics now aggregate Copilot cloud agent active user counts

3. Nuevas métricas de fusión para PRs revisados por Copilot

La API de métricas de uso incorpora dos nuevos campos: total de PRs fusionados revisados por Copilot y tiempo medio hasta la fusión para esos PRs. Combinados con las métricas existentes de PRs creados por el agente, ofrecen visibilidad completa del ciclo de vida de los pull requests.

→ Changelog: Copilot-reviewed pull request merge metrics now in the usage metrics API

4. La actividad del CLI se integra en los totales y desgloses de métricas de uso

Los totales de nivel superior de la API (actividad de código, líneas añadidas/eliminadas) incluyen ahora la actividad del CLI junto a la del IDE, que además aparece como feature=copilot_cli en los desgloses dimensionales. Los paneles que asumían datos únicamente del IDE verán incrementar sus totales.

→ Changelog: Copilot CLI activity now included in usage metrics totals and feature breakdowns

5. Las herramientas de validación del agente en la nube son ahora un 20% más rápidas

El agente en la nube ejecuta ahora en paralelo CodeQL, el Advisory Database, secret scanning y la revisión de código de Copilot, reduciendo el tiempo de validación en un 20%. La calidad es la misma; el agente solicita la revisión humana más rápidamente.

→ Changelog: Copilot cloud agent's validation tools are now 20% faster
