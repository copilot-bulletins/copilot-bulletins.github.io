🏆 Top 5 de actualizaciones

1. Soporte a largo plazo para GPT-5.3-Codex en GitHub Copilot

GitHub garantiza ahora 12 meses de disponibilidad para los modelos designados LTS, comenzando con GPT-5.3-Codex (disponible hasta el 4 de febrero de 2027). Las organizaciones obtienen la estabilidad necesaria para completar sus procesos internos de revisión de seguridad sin el riesgo de interrupciones por cambios de modelo.

→ Changelog: GPT-5.3-Codex long-term support in GitHub Copilot

2. Las métricas de uso de Copilot ahora resuelven la selección automática de modelos

La actividad que antes aparecía como "Auto" en los informes de uso ahora muestra el modelo real utilizado. Los administradores obtienen visibilidad completa del consumo de modelos a nivel de empresa, organización y usuario, lo que facilita el cumplimiento normativo y refuerza la capacidad de auditoría.

→ Changelog: Copilot usage metrics now resolve auto model selection to actual models

3. Configura las herramientas de validación del agente de programación

Los administradores de repositorio controlan ahora qué herramientas de validación ejecuta el agente de programación: CodeQL, advisory database, análisis de secretos y revisión de código. La configuración está disponible en los ajustes del repositorio, bajo Copilot > Agente de programación, sin necesidad de licencia GitHub Advanced Security.

→ Changelog: Configure Copilot coding agent's validation tools

4. Sigue cada commit del agente de programación hasta sus logs de sesión

Cada commit generado por el agente ahora incluye un trailer Agent-Logs-Url con enlace directo a los logs de sesión completos. Esto facilita la trazabilidad del código generado por IA y permite a los revisores auditar el razonamiento del agente durante las revisiones de código.

→ Changelog: Trace any Copilot coding agent commit to its session logs

5. Gemini 3.1 Pro ya disponible en JetBrains, Xcode y Eclipse

Gemini 3.1 Pro está en vista previa pública para todos los planes de Copilot, ahora también en JetBrains IDEs, Xcode y Eclipse. Los administradores de Copilot Business y Enterprise deben activar la política correspondiente en sus ajustes de organización para que los equipos puedan seleccionar el modelo.

→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse
