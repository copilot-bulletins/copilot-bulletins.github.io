Biweekly #1 · Del 10 al 24 de marzo de 2026 · Admins Edition

🎯 El hilo de esta edición

Estas dos semanas, GitHub ha intensificado las capacidades de observabilidad y control para administradores: métricas de modelos más precisas, trazabilidad completa de commits del agente de codificación, nuevas métricas de uso del CLI a nivel organizacional y mayor control sobre las herramientas de validación del agente. La plataforma avanza hacia una gobernanza más sólida del desarrollo asistido por IA.

⚠️ Atención

  • Cierre de la API de métricas de Copilot heredada — Deadline: 2 de abril de 2026 — La API de métricas de Copilot queda fuera de servicio. Migra tus flujos de trabajo a los nuevos endpoints de métricas de uso de Copilot antes de la fecha límite.
  • GPT-5.3-Codex se convierte en el modelo base de Copilot Business y Enterprise — Deadline: 17 de mayo de 2026 — Todas las organizaciones Business y Enterprise serán migradas automáticamente al nuevo modelo base con multiplicador de 1x de solicitudes premium. Revisa tus políticas de modelos para anticipar el impacto.

🚀 Pasa a la acción

  • Migra tus integraciones antes del 2 de abril de 2026 a los nuevos endpoints de métricas de uso de Copilot para evitar interrupciones.
  • Revisa el impacto del cambio de modelo base a GPT-5.3-Codex antes del 17 de mayo: consulta la lista de modelos disponibles y actualiza tus políticas de modelos.
  • Configura qué herramientas de validación ejecuta el agente de codificación desde la sección Copilot > Coding agent en la configuración del repositorio.
  • Habilita el agente de codificación para tu organización y verifica que los registros de sesión sean accesibles para auditorías de código generado por IA.
  • ¿Preguntas o necesitas apoyo? Contacta con tu equipo de GitHub, estamos aquí para ayudarte.

🏆 Top 5 de actualizaciones

1. Métricas de uso: los modelos reales al descubierto

La función "Auto" del selector de modelos ya no aparece como categoría genérica en los informes de métricas. Copilot mapea ahora el modelo real usado en cada sesión, permitiendo datos precisos de consumo por modelo en informes de empresa, organización y usuario — un avance clave para auditorías y políticas de gobernanza de IA.

→ Changelog: Copilot usage metrics now resolve auto model selection to actual models

2. GPT-5.3-Codex: soporte a largo plazo y nuevo modelo base

GPT-5.3-Codex se convierte en el primer modelo LTS de Copilot, con disponibilidad garantizada hasta el 4 de febrero de 2027. A partir del 17 de mayo de 2026, reemplazará a GPT-4.1 como modelo base en Copilot Business y Enterprise con multiplicador de 1x de solicitudes premium. Los equipos que necesiten más tiempo para su proceso de revisión interno deben contactar con su equipo de cuenta.

→ Changelog: GPT-5.3-Codex long-term support in GitHub Copilot

3. Trazabilidad de commits del agente de codificación

Cada commit generado por el agente de codificación Copilot incluye ahora el campo Agent-Logs-Url en el mensaje de commit, con un enlace permanente al registro de la sesión. Permite auditar cambios generados por IA y rastrear las decisiones del agente durante revisiones de código o investigaciones forenses posteriores.

→ Changelog: Trace any Copilot coding agent commit to its session logs

4. Métricas de uso incluyen actividad del CLI de Copilot a nivel organizacional

Las métricas de uso de Copilot a nivel organizacional ahora incluyen datos de actividad del CLI de GitHub Copilot. Los administradores pueden visualizar el uso del CLI junto a los datos de chat e IDE, obteniendo un panorama completo de adopción que enriquece los informes de gobernanza y cumplimiento normativo.

→ Changelog: Copilot usage metrics now includes organization-level GitHub Copilot CLI activity

5. Control sobre las herramientas de validación del agente de codificación

Los administradores de repositorios pueden ahora configurar qué herramientas de validación ejecuta el agente de codificación Copilot: CodeQL, GitHub Advisory Database, análisis de secretos y revisión de código de Copilot. Todas activas por defecto y gratuitas, sin necesidad de licencia de GitHub Advanced Security, con la opción de desactivar herramientas específicas cuando sea necesario.

→ Changelog: Configure Copilot coding agent's validation tools

📌 También en esta edición

  • More visibility into Copilot coding agent sessions — Mejoras en los registros de sesión: visibilidad de pasos de configuración integrados, herramientas personalizadas de entorno y actividad de subagentes.
  • Optionally skip approval for Copilot coding agent Actions workflows — Nueva configuración de repositorio para omitir la aprobación humana previa en los flujos de trabajo de Actions del agente.
