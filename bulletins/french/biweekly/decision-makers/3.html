🏆 Top 5 des mises à jour

1. Résidence des données Copilot en US + UE et conformité FedRAMP disponibles

GitHub Copilot garantit désormais que tout traitement d'inférence reste en régions US ou UE selon votre configuration. Les clients gouvernementaux américains bénéficient également de la certification FedRAMP Moderate. Une avancée décisive pour les secteurs financier, de la santé et du secteur public.

→ Changelog: Copilot data residency in US + EU and FedRAMP compliance now available

2. Mesurer l'impact de la revue de code Copilot sur les délais de livraison

L'API de métriques intègre deux nouveaux indicateurs : pull requests fusionnées après revue Copilot et délai médian de fusion. Ces données permettent de comparer concrètement l'efficacité des cycles de développement avec et sans IA, facilitant la démonstration du ROI.

→ Changelog: Copilot-reviewed pull request merge metrics now in the usage metrics API

3. Suivi de l'adoption de l'agent cloud Copilot dans les métriques d'utilisation

Les rapports d'utilisation au niveau entreprise et organisation incluent désormais le nombre d'utilisateurs actifs de l'agent cloud Copilot (quotidien, hebdomadaire, mensuel). Fini le suivi manuel : décideurs et DSI disposent d'une vue consolidée de l'adoption de l'IA agentique au sein de leurs équipes.

→ Changelog: Copilot usage metrics now aggregate Copilot cloud agent active user counts

4. Les outils de validation de l'agent cloud Copilot sont 20 % plus rapides

L'agent cloud Copilot exécute désormais en parallèle ses outils de validation — CodeQL, détection de secrets, revue de code — réduisant le temps de validation de 20 %. Résultat : des tâches bouclées plus rapidement, sans compromis sur la qualité ou la sécurité.

→ Changelog: Copilot cloud agent's validation tools are now 20% faster

5. Sélection de modèles pour les agents Claude et Codex sur github.com

Les agents de codage tiers Claude et Codex, inclus dans votre abonnement Copilot, permettent désormais de choisir le modèle IA pour chaque tâche (Claude Opus 4.6, GPT-5.4...). Une illustration concrète de la stratégie multi-fournisseurs de GitHub pour rester à la pointe.

→ Changelog: Model selection for Claude and Codex agents on github.com
