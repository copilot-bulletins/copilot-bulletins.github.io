🏆 Top 5 des mises à jour

1. Résidence des données et conformité FedRAMP pour Copilot

GitHub Copilot prend désormais en charge la résidence des données pour les régions US et UE, avec conformité FedRAMP Moderate pour les clients gouvernementaux américains. Toutes les fonctionnalités GA sont couvertes : suggestions, chat, cloud agent, code review. Un catalyseur décisif pour l'adoption dans les secteurs réglementés.

→ Changelog: Copilot data residency in US + EU and FedRAMP compliance now available

2. Choix du modèle pour les agents tiers Claude et Codex

Les utilisateurs peuvent désormais sélectionner le modèle (Anthropic ou OpenAI) lors de l'utilisation des agents Claude et Codex sur github.com, comme pour Copilot cloud agent. GitHub se positionne en plateforme neutre rassemblant les meilleurs modèles du marché, sans dépendance à un seul fournisseur.

→ Changelog: Model selection for Claude and Codex agents on github.com

3. "Rubber Duck" : revue croisée entre familles de modèles

Copilot CLI introduit un agent de revue secondaire, issu d'une autre famille de modèles, pour évaluer plans et implémentations avant exécution. Les benchmarks montrent que Sonnet + Rubber Duck comble 74,7 % de l'écart de performance avec Opus seul — moins d'erreurs coûteuses sur les tâches complexes.

→ Blog: GitHub Copilot CLI combines model families for a second opinion

4. Résoudre les conflits de fusion en trois clics

Un nouveau bouton "Fix with Copilot" sur github.com permet de déléguer automatiquement la résolution des conflits de fusion à Copilot cloud agent, qui vérifie le build, exécute les tests et pousse les corrections. Un gain de temps direct et mesurable pour les équipes de développement.

→ Changelog: Fix merge conflicts in three clicks with Copilot cloud agent

5. Piloter Copilot CLI depuis le web ou mobile

Copilot CLI prend désormais en charge les sessions à distance : les développeurs peuvent superviser et diriger une session CLI active depuis n'importe quel navigateur ou l'application mobile GitHub. Idéal pour les tâches longues déléguées à l'agent, sans contraindre le développeur à rester devant son poste.

→ Changelog: Remote control CLI sessions on web and mobile in public preview
