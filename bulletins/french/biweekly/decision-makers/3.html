🏆 Top 5 des mises à jour

1. Un premier contrat de confiance : les modèles LTS pour les entreprises

GitHub introduit les modèles à support long terme (LTS) pour Copilot Business et Enterprise. GPT-5.3-Codex, premier modèle LTS, est garanti disponible pendant 12 mois (jusqu'au 4 février 2027). Cet engagement donne aux équipes de sécurité le temps nécessaire pour leurs revues internes, sans craindre des changements de modèle non planifiés.

→ Changelog: GPT-5.3-Codex long-term support in GitHub Copilot

2. L'agent de code Copilot démarre 50 % plus vite

Le temps de démarrage de l'agent de code Copilot a été réduit de moitié. Résultat : les équipes obtiennent des pull requests plus rapidement et peuvent itérer en boucle courte avec Copilot sur les revues de code. Un gain direct de productivité à l'échelle de l'organisation.

→ Changelog: Copilot coding agent now starts work 50% faster

3. GPT-5.4 mini disponible en général dans Copilot

GPT-5.4 mini — le dernier modèle rapide d'OpenAI — est désormais en disponibilité générale dans GitHub Copilot. Selon les premiers tests, c'est le modèle mini le plus performant d'OpenAI à ce jour, particulièrement efficace pour l'exploration de code. Il élargit la palette de choix pour vos équipes, avec un multiplicateur de 0,33x.

→ Changelog: GPT-5.4 mini is now generally available for GitHub Copilot

4. Gemini 3.1 Pro disponible sur JetBrains, Xcode et Eclipse

Gemini 3.1 Pro est désormais accessible en préversion publique dans tous les environnements Copilot, y compris JetBrains, Xcode et Eclipse. Les organisations peuvent l'activer via leurs paramètres de politique, offrant aux développeurs un choix de modèle multi-éditeurs adapté à leurs flux de travail.

→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

5. Métriques d'usage : la sélection automatique de modèle est désormais lisible

Lorsque la sélection automatique de modèle est activée, les métriques d'utilisation Copilot affichaient une ligne générique « Auto ». Elles résolvent désormais le nom réel du modèle utilisé, donnant aux administrateurs une vue complète et précise pour leurs rapports de conformité et leurs audits internes.

→ Changelog: Copilot usage metrics now resolve auto model selection to actual models
