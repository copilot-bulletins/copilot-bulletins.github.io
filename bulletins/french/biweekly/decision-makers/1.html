🏆 Top 5 des mises à jour

1. 60 millions de revues de code Copilot : l'IA devenue standard qualité

L'utilisation de Copilot code review a été multipliée par 10 depuis son lancement. Elle représente désormais une revue de code sur cinq sur GitHub, avec plus de 12 000 organisations en automatique. WEX rapporte environ 30 % de code supplémentaire livré grâce à Copilot.

→ Blog: 60 million Copilot code reviews and counting

2. Le Copilot SDK ouvre l'exécution agentique à vos applications

Le GitHub Copilot SDK, disponible en preview publique, permet d'intégrer le moteur de planification et d'exécution de Copilot directement dans vos logiciels. Les applications peuvent désormais déléguer des tâches complexes à des agents au-delà de l'IDE, ouvrant un nouveau paradigme d'automatisation.

→ Blog: The era of "AI as text" is over. Execution is the new interface.

3. Sécurité des Agentic Workflows : des agents sans accès aux secrets

GitHub détaille l'architecture de sécurité des Agentic Workflows : isolation en conteneurs, agents sans accès aux secrets, écriture vérifiée et journalisation complète. Un modèle de menace conçu pour les entreprises qui veulent déployer l'IA dans leurs pipelines CI/CD en toute confiance.

→ Blog: Under the hood: Security architecture of GitHub Agentic Workflows

4. Capacités agentiques en disponibilité générale dans les IDE JetBrains

Les agents personnalisés, sub-agents et plan agent sont désormais GA dans JetBrains. Les agent hooks arrivent en preview, et le support de AGENTS.md facilite la personnalisation. Cette parité renforce Copilot comme solution multi-IDE pour les organisations.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

5. Coding agent : approbation optionnelle des workflows Actions

Les administrateurs de dépôts peuvent désormais autoriser l'exécution automatique des workflows GitHub Actions déclenchés par le coding agent, sans approbation humaine préalable. Ce paramètre accélère la boucle de retour et augmente la vélocité des équipes utilisant l'agent.

→ Changelog: Optionally skip approval for Copilot coding agent Actions workflows
