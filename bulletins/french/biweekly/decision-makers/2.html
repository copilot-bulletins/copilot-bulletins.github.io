🏆 Top 5 des mises à jour

1. Un premier modèle LTS pour une IA d'entreprise prévisible

GitHub annonce GPT-5.3-Codex comme premier modèle à support long terme (LTS) : disponibilité garantie 12 mois pour Copilot Business et Enterprise. Les organisations disposent désormais d'une fenêtre stable pour leurs processus d'approbation internes, réduisant le risque lié aux transitions de modèles non planifiées.

→ Changelog: GPT-5.3-Codex long-term support in GitHub Copilot

2. L'agent de codage Copilot : 50 % plus rapide, plus efficace

Deux améliorations majeures cette quinzaine : l'agent de codage démarre désormais 50 % plus rapidement, et une recherche sémantique de code lui permet de compléter les tâches 2 % plus vite sans perte de qualité. Des gains directs et mesurables sur la vélocité des équipes de développement.

Sources:
→ Changelog: Copilot coding agent now starts work 50% faster
→ Changelog: Copilot coding agent works faster with semantic code search

3. GPT-5.4 mini : le modèle rapide d'OpenAI désormais disponible

GPT-5.4 mini, version rapide du modèle agentique GPT-5.4 d'OpenAI, est maintenant en disponibilité générale pour tous les plans Copilot (Pro, Pro+, Business, Enterprise). Performances accrues sur l'exploration de code et temps de réponse minimal — un atout pour les organisations cherchant à maximiser la productivité sans compromis sur les coûts.

→ Changelog: GPT-5.4 mini is now generally available for GitHub Copilot

4. Gemini 3.1 Pro étendu aux IDEs JetBrains, Xcode et Eclipse

Gemini 3.1 Pro est désormais disponible en préversion publique dans JetBrains, Xcode et Eclipse, en complément de github.com, GitHub Mobile et VS Code. Cette expansion illustre la stratégie multi-fournisseurs de GitHub : offrir le meilleur modèle selon le contexte, sans dépendance à un seul acteur IA.

→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

5. Traçabilité complète des commits générés par l'agent de codage

Chaque commit produit par l'agent de codage Copilot inclut désormais un lien direct vers les journaux de session correspondants. Les responsables peuvent retracer chaque décision de l'IA, faciliter les audits de code et renforcer la gouvernance des déploiements IA en entreprise.

→ Changelog: Trace any Copilot coding agent commit to its session logs
