Biweekly #2 · Du 23 mars au 6 avril 2026 · Decision-Makers Edition

🎯 Le fil de cette édition

Ces deux dernières semaines marquent un tournant stratégique pour GitHub Copilot : la plateforme s'affirme comme une couche d'IA d'entreprise extensible et gouvernable. Le lancement du SDK Copilot, l'expansion de l'agent cloud au-delà des pull requests et les nouveaux contrôles organisationnels dessinent la vision de GitHub : une IA systématique, auditable et intégrée au cœur de chaque organisation.

⚠️ Attention

  • GPT-5.1 Codex, GPT-5.1-Codex-Max et GPT-5.1-Codex-Mini dépréciés — Action requise — Ces modèles ont été retirés le 1er avril 2026. Migrez vos workflows vers GPT-5.3-Codex et vérifiez les politiques de modèles via les paramètres Copilot de vos modèles pris en charge.
  • Dépréciation annoncée de Claude Sonnet 4 dans GitHub Copilot — Délai : 1er mai 2026 — Planifiez la migration vers Claude Sonnet 4.6 avant cette date et activez le nouveau modèle via les politiques de modèles dans vos paramètres Copilot.

🚀 Passez à l'action

  • Activez le Copilot cloud agent pour votre organisation via la documentation d'administration Copilot cloud agent.
  • Définissez des instructions personnalisées à l'échelle de votre organisation dans les paramètres Copilot > Instructions personnalisées.
  • Configurez le runner par défaut et le pare-feu pour le cloud agent dans les paramètres d'organisation Copilot > Cloud agent.
  • Explorez le SDK Copilot pour intégrer les capacités agentiques de Copilot dans vos propres plateformes et workflows internes.
  • Des questions ou besoin d'accompagnement ? Contactez votre équipe GitHub, nous sommes là pour vous aider.

🏆 Top 5 des mises à jour

1. Copilot SDK en préversion publique : l'IA agentique au service de votre plateforme

Le SDK Copilot permet d'intégrer les capacités agentiques de GitHub Copilot directement dans vos applications, workflows et services internes. Disponible en cinq langages (Node.js, Python, Go, .NET, Java), il expose le même moteur d'agent qui propulse le Copilot cloud agent. Support BYOK pour OpenAI, Azure AI Foundry et Anthropic inclus dès le départ.

→ Changelog: Copilot SDK in public preview

2. Copilot cloud agent : recherche, planification et code sans contrainte de PR

L'agent Copilot cloud n'est plus limité à la création de pull requests. Il peut désormais explorer votre base de code, générer un plan d'implémentation pour validation, puis coder sur une branche sans ouvrir de PR immédiatement. Une évolution majeure vers un cycle de développement entièrement piloté par l'IA.

→ Changelog: Research, plan, and code with Copilot cloud agent

3. Instructions personnalisées d'organisation désormais en disponibilité générale

Disponibles depuis avril 2025 en préversion, les instructions personnalisées d'organisation GitHub Copilot passent en disponibilité générale. Les administrateurs Copilot Business et Enterprise peuvent définir des directives globales appliquées automatiquement sur GitHub.com, lors des revues de code Copilot et via le Copilot cloud agent.

→ Changelog: Copilot organization custom instructions are generally available

4. Gouvernance organisationnelle du Copilot cloud agent : runner et pare-feu

Les administrateurs d'organisation contrôlent désormais de manière centralisée le runner GitHub Actions du Copilot cloud agent (standard, large ou auto-hébergé) et la politique de pare-feu réseau. Ces paramètres peuvent être imposés à tous les dépôts, facilitant un déploiement à grande échelle conforme aux exigences de sécurité.

Sources:
→ Changelog: Organization runner controls for Copilot cloud agent
→ Changelog: Organization firewall settings for Copilot cloud agent

5. Copilot cloud agent signe cryptographiquement ses commits

Les commits effectués par le Copilot cloud agent apparaissent désormais comme « Verified » sur GitHub. Cette signature cryptographique garantit leur authenticité et leur intégrité, et permet à l'agent d'opérer dans les dépôts soumis à la règle « Require signed commits » — une exigence fréquente dans les environnements régulés.

→ Changelog: Copilot cloud agent signs its commits
