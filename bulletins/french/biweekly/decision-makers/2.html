🎯 Le fil de cette édition

Cette quinzaine marque une accélération vers la maturité enterprise de GitHub Copilot. Entre la stabilité garantie par le premier modèle LTS, le gain de performance de 50 % de l'agent de codage et le renforcement de la visibilité sur les métriques, GitHub pose les bases d'une infrastructure d'IA fiable et gouvernable — au-delà du simple outil de productivité.

🚀 Passez à l'action

  • Anticipez la transition : activez GPT-5.3-Codex dans vos paramètres de modèles Copilot avant le 17 mai 2026 pour éviter toute interruption liée aux processus internes de validation.
  • Donnez à vos équipes accès aux nouveaux modèles Gemini 3.1 Pro et GPT-5.4 mini en les activant depuis la page des modèles Copilot de vos paramètres Enterprise ou Business.

  • Des questions ou besoin d'accompagnement ? Contactez votre équipe GitHub, nous sommes là pour vous aider.

🏆 Top 5 des mises à jour

1. GPT-5.3-Codex en support à long terme (LTS) dans GitHub Copilot

GitHub lance son premier modèle LTS en partenariat avec OpenAI : GPT-5.3-Codex sera disponible au minimum 12 mois pour les plans Business et Enterprise (jusqu'au 4 février 2027). Les entreprises disposent ainsi d'un horizon de stabilité prévisible pour leurs processus internes de validation et de conformité, sans risque de dépréciation surprise.

→ Changelog: GPT-5.3-Codex long-term support in GitHub Copilot

2. L'agent de codage Copilot démarre 50 % plus vite

GitHub a optimisé l'agent de codage Copilot pour réduire de moitié son temps de démarrage. Les cycles d'itération sur les pull requests s'accélèrent, ce qui amplifie directement le retour sur investissement de l'agent — que ce soit pour les équipes produit ou les développeurs à la cadence de livraison soutenue.

→ Changelog: Copilot coding agent now starts work 50% faster

3. Gemini 3.1 Pro disponible dans JetBrains, Xcode et Eclipse

Le modèle Gemini 3.1 Pro de Google est désormais accessible en préversion publique sur tous les IDEs majeurs. GitHub Copilot couvre maintenant l'ensemble des plateformes de développement — VS Code, Visual Studio, JetBrains, Xcode et Eclipse — avec un choix de modèles élargi, renforçant sa position de plateforme d'IA multi-modèle et multi-IDE.

→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

4. GPT-5.4 mini en disponibilité générale dans Copilot

Le dernier modèle mini haute performance d'OpenAI est désormais disponible dans Copilot pour tous les plans. GPT-5.4 mini offre le meilleur temps de première réponse de la gamme mini et excelle dans l'exploration de bases de code — avec un multiplicateur de requêtes premium de seulement 0,33x, ce qui en fait une option rapide et économique pour les grandes équipes.

→ Changelog: GPT-5.4 mini is now generally available for GitHub Copilot

5. Les métriques d'utilisation Copilot résolvent la sélection automatique de modèle

Les métriques d'utilisation Copilot identifient désormais le modèle réel utilisé lorsque la sélection automatique est activée. Fini les données agrégées sous l'étiquette générique « Auto » — les responsables disposent d'une visibilité complète par modèle pour appuyer leurs décisions de gouvernance et leurs audits de conformité.

→ Changelog: Copilot usage metrics now resolve auto model selection to actual models
