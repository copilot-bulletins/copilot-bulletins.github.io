Biweekly #2 · Du 10 au 24 mars 2026 · Devs Edition

🎯 Le fil de cette édition

Ces deux dernières semaines, les améliorations convergent autour d'un objectif : rendre l'agent de codage Copilot plus rapide, plus transparent et plus configurable. Le choix de modèles s'élargit également, avec GPT-5.4 mini en disponibilité générale et Gemini 3.1 Pro dans davantage d'IDE. Les développeurs profitent d'un cycle de feedback plus court et d'un meilleur contrôle sur leurs workflows.

🚀 Passez à l'action

  • Activez l'agent de codage Copilot pour déléguer des tâches de développement en arrière-plan.
  • Explorez les nouveaux modèles disponibles dans le sélecteur et trouvez celui qui correspond à votre cas d'usage.
  • Demandez une révision de code Copilot sans quitter votre terminal avec gh pr edit --add-reviewer @copilot``.
  • Des questions ou besoin d'aide ? Contactez votre équipe GitHub, nous sommes là pour vous.

🏆 Top 5 des mises à jour

1. L'agent de codage Copilot démarre 50% plus vite

L'agent de codage Copilot démarre désormais 50% plus vite. Assignez une issue, ouvrez un PR depuis l'onglet Agents ou mentionnez @copilot dans un commentaire : le cycle de feedback est nettement raccourci. Un gain concret pour déléguer des tâches en continu.

→ Changelog: Copilot coding agent now starts work 50% faster

2. Améliorations majeures des capacités agentiques dans JetBrains

Les agents personnalisés, sous-agents et l'agent de planification passent en GA dans JetBrains. Hooks d'agent (preview), auto-approve MCP, support des fichiers AGENTS.md/CLAUDE.md et sélection automatique de modèle complètent la mise à jour. Un bond majeur pour les équipes JetBrains.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

3. GPT-5.4 mini désormais en disponibilité générale

GPT-5.4 mini, la version rapide du modèle agentique GPT-5.4 d'OpenAI, est en disponibilité générale. C'est le modèle mini le plus performant d'OpenAI à ce jour, avec le temps de premier token le plus rapide et une excellente exploration de codebase.

→ Changelog: GPT-5.4 mini is now generally available for GitHub Copilot

4. Gemini 3.1 Pro disponible dans JetBrains, Xcode et Eclipse

Gemini 3.1 Pro est en preview publique dans JetBrains IDEs, Xcode et Eclipse, en plus de GitHub.com, Mobile, VS Code et Visual Studio. Accessible via le sélecteur de modèle en modes chat, ask et edit pour les plans Pro, Pro+, Business et Enterprise.

→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

5. Révision de code Copilot depuis le GitHub CLI

Plus besoin de passer par le navigateur : ajoutez Copilot comme relecteur avec gh pr edit --add-reviewer @copilot`` ou via les prompts interactifs de gh pr create. Compatible avec gh CLI v2.88.0+. La sélection de reviewers est aussi plus rapide grâce à la recherche à la volée.

→ Changelog: Request Copilot code review from GitHub CLI

📌 Également dans cette édition

  • Copilot coding agent works faster with semantic code search — L'agent utilise désormais la recherche sémantique pour trouver du code pertinent sans correspondance exacte, réduisant le temps d'exécution de 2%.
  • Explore a repository using Copilot on the web — Parcourez l'arborescence de fichiers d'un dépôt en parallèle du chat Copilot sur GitHub.com ; les fichiers sélectionnés s'ajoutent automatiquement comme références.
  • More visibility into Copilot coding agent sessions — Les logs de session affichent maintenant les étapes de setup, les configurations personnalisées et l'activité des sous-agents.
