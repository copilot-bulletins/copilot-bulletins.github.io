🎯 Le fil de cette édition

Ces deux dernières semaines, l'agent de codage Copilot a franchi un nouveau cap : démarrage 50 % plus rapide, commits traçables jusqu'aux logs de session, et visibilité accrue sur ses actions internes. Gemini 3.1 Pro s'étend désormais à JetBrains, Xcode et Eclipse. L'IA agentique s'intègre pleinement dans le flux de travail quotidien des développeurs.

🚀 Passez à l'action

  • Accédez à Gemini 3.1 Pro via le sélecteur de modèles Copilot dans votre IDE ou sur GitHub.com
  • Suivez les sessions de l'agent de codage en temps réel depuis les logs de session Copilot
  • Activez l'agent de codage depuis la page Politiques de votre organisation
  • Des questions ? Contactez votre équipe GitHub, nous sommes là pour vous aider.

🏆 Top 5 des mises à jour

1. L'agent de codage Copilot démarre 50 % plus vite

GitHub a optimisé l'agent de codage Copilot pour réduire son temps de démarrage de moitié. Que vous déléguiez une issue, une pull request ou un prompt direct, l'agent prend en main votre tâche bien plus rapidement. Le feedback loop avec @copilot est sensiblement plus serré.

→ Changelog: Copilot coding agent now starts work 50% faster

2. Gemini 3.1 Pro disponible dans JetBrains, Xcode et Eclipse

Gemini 3.1 Pro passe en préversion publique dans JetBrains, Xcode et Eclipse — en plus de VS Code, Visual Studio et GitHub.com. Disponible en modes chat, agent et edit via le sélecteur de modèles. Les admins Copilot Business/Enterprise doivent l'activer dans les paramètres Copilot.

→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

3. Plus de visibilité dans les sessions de l'agent de codage

Les logs de session s'enrichissent : étapes de démarrage désormais visibles, sorties des scripts d'environnement personnalisés affichées, sous-agents affichés de façon condensée avec un aperçu en temps réel. Vous suivez exactement ce que fait l'agent sans quitter votre interface de review.

→ Changelog: More visibility into Copilot coding agent sessions

4. Tracez les commits de l'agent jusqu'aux logs de session

Chaque commit produit par l'agent de codage Copilot inclut désormais un trailer Agent-Logs-Url pointant vers les logs de la session. Lors d'une code review, en un clic, retracez le raisonnement de l'agent et comprenez pourquoi un changement a été effectué.

→ Changelog: Trace any Copilot coding agent commit to its session logs

5. Surveillez les logs de l'agent en direct dans Raycast

L'extension GitHub Copilot pour Raycast permet désormais de suivre les logs de l'agent de codage en temps réel, depuis n'importe où sur macOS ou Windows. Ouvrez Raycast, sélectionnez View Tasks, choisissez votre session — et gardez un œil sur l'avancement sans ouvrir GitHub.

→ Changelog: Monitor Copilot coding agent logs live in Raycast
