🏆 Top 5 des mises à jour

1. Contrôle à distance des sessions CLI depuis le web et le mobile

Le Copilot CLI n'est plus limité à votre terminal local. Avec copilot --remote, démarrez une session CLI et suivez-la ou pilotez-la depuis github.com ou GitHub Mobile, avec synchronisation en temps réel. Idéal pour les longues tâches en arrière-plan ou pour travailler depuis un autre appareil.

→ Changelog: Remote control CLI sessions on web and mobile in public preview

2. Résolution des conflits de fusion en trois clics avec l'agent cloud

Un nouveau bouton Fix with Copilot apparaît directement sur vos pull requests en cas de conflit. Cliquez, validez, et Copilot analyse les changements, résout les conflits, s'assure que les tests passent, puis vous demande de relire. Mentionnez aussi @copilot dans un commentaire pour d'autres corrections.

→ Changelog: Fix merge conflicts in three clicks with Copilot cloud agent

3. Rubber Duck : une revue croisée entre modèles dans le CLI

En mode expérimental (/experimental), Rubber Duck mobilise un second modèle d'une famille différente pour examiner vos plans et implémentations. Claude Sonnet + Rubber Duck (GPT-5.4) comble 74,7 % de l'écart de performance entre Sonnet et Opus sur les tâches complexes multi-fichiers.

→ Blog: GitHub Copilot CLI combines model families for a second opinion

4. Sélection de modèle pour les agents tiers Claude et Codex

Comme pour l'agent cloud Copilot, vous pouvez désormais choisir le modèle lors du démarrage d'une tâche avec les agents Claude (Anthropic) ou Codex (OpenAI) sur github.com : Claude Opus/Sonnet 4.5–4.6, GPT-5.2-Codex à 5.4. L'accès est inclus dans votre abonnement Copilot existant.

→ Changelog: Model selection for Claude and Codex agents on github.com

5. Les outils de validation de l'agent cloud sont 20 % plus rapides

CodeQL, GitHub Advisory Database, secret scanning et Copilot code review s'exécutent désormais en parallèle lors des tâches de l'agent cloud. Résultat : une réduction de 20 % du temps de validation, sans compromis sur la qualité du code livré.

→ Changelog: Copilot cloud agent's validation tools are now 20% faster
