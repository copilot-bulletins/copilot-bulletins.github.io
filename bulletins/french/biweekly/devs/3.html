Biweekly #3 · Du 10 au 24 mars 2026 · Devs Edition

🎯 Le fil de cette édition

Les deux dernières semaines ont été marquées par une accélération notable de l'agent de codage Copilot : démarrage 50 % plus rapide, recherche sémantique de code, meilleure observabilité des sessions et outils de validation configurables. En parallèle, GPT-5.4 mini passe en disponibilité générale et Gemini 3.1 Pro s'étend à de nouveaux IDEs. L'agent avance vite, et les outils pour le maîtriser aussi.

⚠️ Attention

  • GPT-5.3-Codex long-term support in GitHub Copilot — Échéance : 17 mai 2026 — GPT-5.3-Codex remplacera automatiquement GPT-4.1 comme modèle de base pour toutes les organisations Copilot Business et Enterprise. Vérifiez l'impact avec votre équipe et consultez la documentation sur les modèles supportés.

🚀 Passez à l'action

  • Activez GPT-5.4 mini dans les paramètres Copilot de votre organisation pour accéder au modèle agentic le plus rapide d'OpenAI — voir la documentation sur les modèles disponibles.
  • Déléguez vos premières tâches à l'agent de codage Copilot directement depuis vos issues GitHub — apprenez à créer une PR avec l'agent.
  • Mettez à jour votre plugin JetBrains Copilot vers la dernière version pour accéder aux agents personnalisés GA, aux agent hooks en preview et à l'auto-approve MCP — voir le changelog.
  • Passez à GitHub CLI v2.88.0 ou supérieur pour demander une revue de code Copilot depuis le terminal — voir la documentation sur Copilot code review :
gh pr edit --add-reviewer `@copilot`
  • Des questions ou besoin d'aide ? Contactez votre équipe GitHub, nous sommes là pour vous accompagner.

🏆 Top 5 des mises à jour

1. Améliorations majeures des capacités agentiques dans JetBrains IDEs

Les agents personnalisés, sous-agents et plan agent sont désormais en disponibilité générale dans JetBrains IDEs. Les agent hooks arrivent en preview publique, l'auto-approve MCP est configurable, et les fichiers d'instructions AGENTS.md et CLAUDE.md sont pris en charge. La sélection automatique de modèle passe aussi en GA. Une mise à jour substantielle pour les équipes JetBrains.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

2. L'agent de codage Copilot démarre 50 % plus vite

Le temps de démarrage de l'agent de codage Copilot a été réduit de moitié. Concrètement, vos pull requests arrivent plus vite et le cycle de feedback lors des itérations avec @copilot est plus court. Aucune configuration requise — l'amélioration est automatique.

→ Changelog: Copilot coding agent now starts work 50% faster

3. GPT-5.4 mini GA et Gemini 3.1 Pro dans plus d'IDEs

GPT-5.4 mini, le modèle agentic rapide d'OpenAI, est maintenant en disponibilité générale pour tous les abonnés Copilot Pro, Pro+, Business et Enterprise. Gemini 3.1 Pro arrive en preview publique dans JetBrains IDEs, Xcode et Eclipse, élargissant vos options au-delà de VS Code et GitHub.com.

Sources:
→ Changelog: GPT-5.4 mini is now generally available for GitHub Copilot
→ Changelog: Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

4. Demandez une revue de code Copilot depuis le terminal

Avec GitHub CLI v2.88.0, ajoutez Copilot comme reviewer directement depuis le terminal avec gh pr edit --add-reviewer @copilot``, sans quitter votre flux de travail. La sélection de reviewers est maintenant basée sur une recherche en temps réel, plus performante dans les grandes organisations. Disponible sur tous les plans incluant Copilot code review.

→ Changelog: Request Copilot code review from GitHub CLI

5. Retracez chaque commit de l'agent vers ses logs de session

Les commits générés par l'agent de codage Copilot incluent désormais un trailer Agent-Logs-Url dans le message de commit, qui pointe directement vers les logs de la session. Idéal pour comprendre les décisions de l'agent lors d'une revue de code ou pour auditer les changements après coup.

→ Changelog: Trace any Copilot coding agent commit to its session logs

  • Copilot coding agent works faster with semantic code search — L'agent utilise désormais la recherche sémantique pour trouver du code pertinent sans se limiter aux correspondances exactes, réduisant le temps de traitement de 2 %.
