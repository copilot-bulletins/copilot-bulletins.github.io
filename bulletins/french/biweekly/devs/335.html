GitHub Copilot Bulletin — Biweekly #1

Du 2 au 16 mars 2026 · Devs Edition

🎯 Le fil de cette édition

Les capacités agentiques de GitHub Copilot franchissent un cap sur tous les environnements. VS Code, JetBrains, CLI et le web gagnent simultanément des hooks, des plugins d'agent, l'auto-approbation MCP et un nouveau modèle GPT-5.4. L'assistant de code se transforme en plateforme d'exécution programmable au service du développeur.

🚀 Passez à l'action

  • Mettez à jour VS Code vers la version 1.110 pour accéder aux hooks, plugins d'agent et outils de navigation agentique.
  • Activez GPT-5.4 dans le sélecteur de modèles — consultez la liste des modèles disponibles.
  • Installez le serveur MCP Figma pour connecter vos designs à votre code dans VS Code.
  • Mettez à jour le plugin JetBrains pour créer des agents personnalisés et configurer des hooks.
  • Des questions ou besoin d'aide ? Contactez votre équipe GitHub, nous sommes là pour vous accompagner.

🏆 Top 5 des mises à jour

1. VS Code v1.110 : les agents passent à la vitesse supérieure

La release de février rend les agents pratiques pour les tâches complexes. Hooks, plugins d'agent, /autoApprove, fork de conversations, outils de navigation automatisée et intégration native de Copilot CLI transforment VS Code en environnement agentique complet.

→ Changelog: GitHub Copilot in Visual Studio Code v1.110 – February release

2. GPT-5.4, le nouveau modèle de référence pour le code

GPT-5.4, le dernier modèle agentique d'OpenAI, est disponible dans GitHub Copilot. Il excelle en raisonnement logique et en exécution de tâches multi-étapes, avec des performances supérieures sur les workflows agentiques et le développement logiciel.

→ Changelog: GPT-5.4 is generally available in GitHub Copilot

3. JetBrains : agents personnalisés, hooks et MCP en GA

Les agents personnalisés, sub-agents et plan agent sont en disponibilité générale dans JetBrains. Les hooks d'agent arrivent en preview publique, avec le support de l'auto-approbation MCP et des fichiers d'instructions AGENTS.md.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

4. Figma + VS Code : le design-to-code bidirectionnel via MCP

Le serveur MCP Figma permet désormais de transformer du code en layers Figma éditables, en plus de générer du code depuis un design. Un workflow connecté pour itérer entre design et implémentation sans quitter VS Code.

→ Changelog: Figma MCP server can now generate design layers from VS Code

5. Code review Copilot depuis le terminal

GitHub CLI v2.88.0 permet de demander une code review à Copilot directement depuis le terminal avec gh pr edit --add-reviewer @copilot``. Plus besoin de basculer vers le navigateur pour obtenir un retour rapide sur vos pull requests.

→ Changelog: Request Copilot code review from GitHub CLI
