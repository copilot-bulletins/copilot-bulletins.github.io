GitHub Copilot Bulletin — Biweekly #1

Du 2 au 16 mars 2026 · Devs Edition

🎯 Le fil de cette édition

Ces deux dernières semaines, GitHub Copilot a considérablement renforcé ses capacités agentiques dans les IDE. VS Code et JetBrains reçoivent des mises à jour majeures qui rendent les agents plus autonomes, personnalisables et contextuellement intelligents. En parallèle, l'arrivée de GPT-5.4 et de nouvelles intégrations MCP élargissent le champ d'action de Copilot au-delà du code.

🚀 Passez à l'action

  • Essayez GPT-5.4 en le sélectionnant dans le sélecteur de modèles de votre IDE ou sur la page des modèles disponibles.
  • Configurez des agent hooks dans votre dépôt pour automatiser vos workflows agentiques — consultez la documentation sur les hooks.
  • Créez des agents personnalisés pour vos tâches récurrentes grâce aux profils d'agents custom.
  • Demandez une revue de code Copilot depuis votre terminal avec gh pr edit --add-reviewer @copilot`` — en savoir plus.
  • Des questions ou besoin d'accompagnement ? Contactez votre équipe GitHub, nous sommes là pour vous aider.

🏆 Top 5 des mises à jour

1. VS Code v1.110 : les agents passent à la vitesse supérieure

Cette release transforme les agents en outils véritablement opérationnels. Hooks d'automatisation, fork de conversations, agent plugins, outils navigateur, compaction de contexte et mémoire partagée entre agents — les workflows longs et complexes deviennent enfin gérables.

→ Changelog: GitHub Copilot in Visual Studio Code v1.110 – February release

2. GPT-5.4 disponible dans GitHub Copilot

Le dernier modèle agentique d'OpenAI est maintenant accessible dans tous les IDE, sur github.com, GitHub Mobile et GitHub CLI. GPT-5.4 excelle dans le raisonnement logique et l'exécution de tâches multi-étapes complexes.

→ Changelog: GPT-5.4 is generally available in GitHub Copilot

3. JetBrains : agents custom, sub-agents et plan agent en GA

Les capacités agentiques de Copilot pour JetBrains franchissent un cap. Agents personnalisés, sub-agents et plan agent sont en disponibilité générale. Les agent hooks arrivent en preview, avec le support d'auto-approve MCP et des fichiers d'instructions AGENTS.md.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

4. Serveur MCP Figma : du design au code et retour

Le serveur MCP Figma permet désormais de générer du code depuis vos designs Figma et d'envoyer vos rendus UI vers Figma comme calques éditables. Un workflow bidirectionnel design-code disponible dans VS Code.

→ Changelog: Figma MCP server can now generate design layers from VS Code

5. Revue de code Copilot depuis le terminal

Vous pouvez maintenant demander une revue de code Copilot directement depuis GitHub CLI avec gh pr edit --add-reviewer @copilot`` ou gh pr create. La sélection de reviewers est aussi plus rapide grâce à un mode recherche.

→ Changelog: Request Copilot code review from GitHub CLI
