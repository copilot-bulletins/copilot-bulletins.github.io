🎯 Le fil de cette édition

Ces deux dernières semaines, GitHub a renforcé le contrôle et la visibilité pour les administrateurs Copilot. Nouveaux filtres de sessions agentiques, paramètre d'approbation des workflows pour le coding agent, activation requise de GPT-5.4, et sélection automatique de modèles respectant les politiques d'entreprise : l'accent est mis sur la gouvernance des capacités IA à l'échelle.

🚀 Passez à l'action

  • Activez GPT-5.4 dans les politiques Copilot de votre entreprise pour le rendre disponible à vos développeurs.
  • Explorez les nouveaux filtres de sessions agentiques dans le panneau AI Controls de votre entreprise.
  • Évaluez le paramètre d'approbation automatique des workflows pour le coding agent dans les paramètres Copilot de vos dépôts.
  • Consultez la documentation sur la sélection automatique de modèles pour comprendre son impact sur vos politiques.
  • Des questions ou besoin d'accompagnement ? Contactez votre équipe GitHub, nous sommes là pour vous aider.

🏆 Top 5 des mises à jour

1. Nouveaux filtres pour superviser l'activité agentique en entreprise

Le panneau AI Controls intègre désormais des filtres par statut, dépôt et utilisateur pour les sessions agentiques. Ces filtres facilitent la découverte et le suivi de l'activité des agents Copilot à travers votre entreprise, renforçant la conformité et la visibilité opérationnelle.

→ Changelog: Discover and manage agent activity with new session filters

2. Approbation optionnelle des workflows Actions pour le coding agent

Un nouveau paramètre de dépôt permet aux administrateurs de désactiver l'approbation humaine avant l'exécution des workflows GitHub Actions déclenchés par le coding agent. Ce compromis entre sécurité et rapidité d'itération offre plus de flexibilité aux équipes, tout en restant désactivé par défaut.

→ Changelog: Optionally skip approval for Copilot coding agent Actions workflows

3. GPT-5.4 disponible — activation requise par les administrateurs

GPT-5.4, le dernier modèle agentique d'OpenAI, est maintenant disponible dans GitHub Copilot. Les administrateurs Business et Enterprise doivent activer la politique GPT-5.4 dans les paramètres Copilot pour que leurs développeurs puissent l'utiliser.

→ Changelog: GPT-5.4 is generally available in GitHub Copilot

4. Sélection automatique de modèles GA dans JetBrains — conforme aux politiques

La sélection automatique de modèles (auto) est désormais en disponibilité générale dans les IDE JetBrains. Point clé pour les administrateurs : auto respecte toutes les politiques de modèles configurées et se limite aux modèles avec un multiplicateur ≤1x, offrant en prime une réduction de 10 % sur les requêtes premium.

→ Changelog: Copilot auto model selection is generally available in JetBrains IDEs

5. Améliorations agentiques majeures dans JetBrains — agent hooks et politiques

Les agents personnalisés, sous-agents et agent plan sont en GA dans JetBrains. Les agent hooks (preview) permettent d'appliquer des politiques et d'automatiser des contrôles à des moments clés des sessions agentiques. Note : les abonnés Business/Enterprise doivent activer la politique « Editor preview features » pour utiliser les hooks.

→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs
