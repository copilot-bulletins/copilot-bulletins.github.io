🏆 Top 5 des mises à jour

1. GPT-5.3-Codex : premier modèle LTS pour les entreprises

Pour les entreprises ayant besoin de stabilité dans leurs processus d'évaluation interne, GitHub lance GPT-5.3-Codex comme premier modèle en support long terme (12 mois). Il devient également le modèle de base pour Copilot Business et Copilot Enterprise à partir du 17 mai 2026.

→ Changelog: GPT-5.3-Codex long-term support in GitHub Copilot

2. Métriques Copilot : le modèle réel remplace la mention « Auto »

Les activités jusqu'ici comptabilisées sous la mention « Auto » s'affichent désormais avec le nom réel du modèle utilisé. Admins et équipes conformité disposent ainsi d'une vue précise de l'utilisation par modèle, aux niveaux entreprise, organisation et utilisateur.

→ Changelog: Copilot usage metrics now resolve auto model selection to actual models

3. Configurer les outils de validation du coding agent

Les administrateurs de dépôt peuvent maintenant activer ou désactiver les outils de validation intégrés du coding agent — CodeQL, secret scanning, revue de code Copilot — directement depuis les paramètres du dépôt, sans licence GitHub Advanced Security requise.

→ Changelog: Configure Copilot coding agent's validation tools

4. Métriques Copilot : l'activité CLI intégrée au niveau organisation

L'activité GitHub Copilot CLI est maintenant visible dans les métriques d'utilisation au niveau organisation. Les admins peuvent suivre comment leurs équipes utilisent Copilot en dehors des IDE et mesurer l'adoption dans les workflows en ligne de commande.

→ Changelog: Copilot usage metrics now includes organization-level GitHub Copilot CLI activity

5. Ignorer optionnellement l'approbation des workflows Actions pour le coding agent

Jusqu'ici, les workflows GitHub Actions ne s'exécutaient pas automatiquement lorsque le coding agent poussait des modifications. Un nouveau paramètre de dépôt permet aux administrateurs de lever cette exigence d'approbation pour accélérer les boucles de validation.

→ Changelog: Optionally skip approval for Copilot coding agent Actions workflows
