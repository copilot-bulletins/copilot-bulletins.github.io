🎯 Le fil de cette édition

Ce mois-ci, GitHub Copilot franchit un cap décisif : l'agent devient programmable. Avec les hooks, les plugins, la mémoire partagée et de nouvelles intégrations (Jira, Figma, CLI), les développeurs peuvent désormais orchestrer des workflows agentiques sur mesure dans VS Code, JetBrains et le web. L'ère du simple assistant de complétion cède la place à celle de l'agent autonome et extensible.

🏆 Top 5 des mises à jour du mois

1. VS Code v1.110 : les agents deviennent programmables

Cette release transforme l'expérience agent dans VS Code. Les hooks permettent d'exécuter du code à chaque étape clé, les plugins d'agents ajoutent des compétences, et la mémoire partagée relie coding agent, code review et CLI. Copilot CLI est désormais intégré nativement.

→ Changelog: GitHub Copilot in Visual Studio Code v1.110 – February release

2. GPT-5.4 arrive dans GitHub Copilot

Le nouveau modèle agentique d'OpenAI est disponible sur tous les plans payants. GPT-5.4 excelle en raisonnement logique et en exécution de tâches multi-étapes complexes. Il est sélectionnable dans VS Code, JetBrains, Visual Studio, Xcode, Eclipse, github.com et le CLI.

→ Changelog: GPT-5.4 is generally available in GitHub Copilot

3. La revue de code Copilot passe à l'architecture agentique

Copilot code review utilise désormais un système agentique pour explorer le dépôt, comprendre l'architecture et fournir des commentaires plus pertinents avec moins de bruit. Vous pouvez aussi demander une review depuis le terminal avec gh pr edit --add-reviewer @copilot``.

Sources:
→ Changelog: Copilot code review now runs on an agentic architecture
→ Changelog: Request Copilot code review from GitHub CLI
→ Blog: 60 million Copilot code reviews and counting

4. JetBrains : agents personnalisés, hooks et sélection automatique de modèle

Les agents personnalisés, sub-agents et plan agent sont désormais GA dans JetBrains. Les agent hooks arrivent en preview publique, le support des fichiers AGENTS.md est ajouté, et la sélection automatique de modèle (auto) est GA avec une remise de 10 % sur les requêtes premium.

Sources:
→ Changelog: Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs
→ Changelog: Copilot auto model selection is generally available in JetBrains IDEs

5. Copilot s'étend : intégration Jira, serveur MCP Figma et Copilot SDK

Copilot coding agent peut maintenant recevoir des issues Jira et créer des PR automatiquement. Le serveur MCP Figma permet un workflow bidirectionnel design↔︎code dans VS Code. Et le Copilot SDK offre un moteur d'exécution agentique programmable pour vos propres applications.

Sources:
→ Changelog: GitHub Copilot coding agent for Jira is now in public preview
→ Changelog: Figma MCP server can now generate design layers from VS Code
→ Blog: The era of "AI as text" is over. Execution is the new interface.
