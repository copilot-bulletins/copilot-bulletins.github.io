Monthly #2 · Avril 2026 · Admins Edition

🎯 Le fil de cette édition

Mars 2026 marque un tournant dans la gouvernance de Copilot pour les administrateurs : GitHub a considérablement étendu les contrôles centralisés de l'agent cloud, enrichi les tableaux de bord de métriques d'adoption et atteint la disponibilité générale des instructions personnalisées à l'échelle de l'organisation. Ces évolutions permettent aux entreprises de déployer Copilot à grande échelle avec visibilité, traçabilité et conformité renforcées.

⚠️ Attention

  • GPT-5.1-Codex, GPT-5.1-Codex-Max et GPT-5.1-Codex-Mini dépréciés — Action requise — Migrez vers GPT-5.3-Codex et vérifiez les stratégies de modèles dans vos paramètres Copilot Enterprise.
  • Dépréciation à venir : Claude Sonnet 4 — Échéance : 1er mai 2026 — Activez Claude Sonnet 4.6 dans vos stratégies de modèles Copilot avant cette date.
  • Gemini 3 Pro déprécié — Action requise — Migrez vers Gemini 3.1 Pro via vos paramètres de stratégie de modèles.

🚀 Passez à l'action

  • Définissez le runner par défaut et verrouillez-le pour toute l'organisation dans les paramètres de runner pour l'agent cloud Copilot.
  • Configurez le pare-feu de l'agent cloud Copilot à l'échelle de l'organisation dans les paramètres du pare-feu de l'agent.
  • Ajoutez des instructions personnalisées pour votre organisation dans les paramètres Copilot de l'organisation.
  • Analysez les métriques d'adoption Copilot pour suivre l'utilisation CLI, de l'agent et la répartition des modèles.
  • Questions ou besoin d'aide ? Contactez votre équipe GitHub, nous sommes là pour vous accompagner.

🏆 Top 5 des mises à jour du mois

1. Gouvernance centralisée de l'agent cloud Copilot au niveau organisation

Les administrateurs d'organisation peuvent désormais définir un runner par défaut pour l'agent cloud Copilot et le verrouiller à l'échelle de l'organisation, configurer le pare-feu de l'agent centralement, et bénéficier de commits signés et vérifiés — renforçant sécurité et conformité du déploiement.

Sources : → Changelog: Organization runner controls for Copilot cloud agent
→ Changelog: Organization firewall settings for Copilot cloud agent
→ Changelog: Copilot cloud agent signs its commits

2. Métriques d'adoption Copilot enrichies : vue complète à l'échelle de l'organisation

Quatre mises à jour successives enrichissent les métriques Copilot : suivi CLI par utilisateur et au niveau organisation, identification des utilisateurs actifs de l'agent, résolution des modèles derrière la sélection automatique — une visibilité d'adoption complète pour planifier et auditer.

Sources : → Changelog: Copilot usage metrics now includes per-user GitHub Copilot CLI activity in organization reports
→ Changelog: Copilot usage metrics now identify active Copilot coding agent users
→ Changelog: Copilot usage metrics now resolve auto model selection to actual models
→ Changelog: Copilot usage metrics now includes organization-level GitHub Copilot CLI activity

3. Instructions personnalisées à l'échelle de l'organisation — disponibilité générale

Les instructions personnalisées à l'échelle de l'organisation passent en disponibilité générale pour Copilot Business et Enterprise. Les administrateurs peuvent définir des consignes applicables à Copilot Chat, la revue de code et l'agent cloud — garantissant une cohérence comportementale dans toute l'organisation.

→ Changelog: Copilot organization custom instructions are generally available

4. Traçabilité et audit renforcés des sessions de l'agent Copilot

L'agent Copilot expose désormais des journaux de session enrichis (étapes de configuration, sous-agents, environnement personnalisé) et intègre un lien permanent vers les logs dans chaque commit — permettant de retracer précisément l'activité de l'agent pour audit et revue de code.

Sources : → Changelog: More visibility into Copilot coding agent sessions
→ Changelog: Trace any Copilot coding agent commit to its session logs

5. Mises à jour importantes : Politique de confidentialité et Conditions d'utilisation

GitHub met à jour sa Politique de confidentialité et ses Conditions d'utilisation. À compter du 24 avril 2026, les données des utilisateurs Copilot Free, Pro et Pro+ pourront servir à l'entraînement de modèles IA, sauf opt-out. Copilot Business et Enterprise ne sont pas affectés.

→ Changelog: Updates to our Privacy Statement and Terms of Service: How we use your data
